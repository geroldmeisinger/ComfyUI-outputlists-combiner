## Spreadsheet OutputList

![Spreadsheet OutputList](SpreadsheetOutputList/SpreadsheetOutputList.png)

(ComfyUI рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ)

рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ OutputList-рЃА рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃў рЃфрЃ«рЃарЃўрЃџрЃўрЃЊрЃљрЃю (`.csv .tsv .ods .xlsx .xls`).
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ `Load any File` рЃюрЃЮрЃЊрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃЉрЃљрЃќрЃўрЃА-64 рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃЌ рЃЕрЃљрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ.
рЃерЃўрЃЊрЃљ рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА *pandas* [read_excel](https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.read_excel.html) рЃЊрЃљ [read_csv](https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.read_csv.html) рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃў рЃфрЃ«рЃарЃўрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЕрЃљрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ.
рЃДрЃЋрЃћрЃџрЃљ рЃАрЃўрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `is_output_list=True` (рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА `­Юїа`) рЃЊрЃљ рЃЏрЃЮрЃЊрЃўрЃА рЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃўрЃЌ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃюрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃўрЃћрЃа.

### рЃерЃћрЃДрЃЋрЃљрЃюрЃћрЃЉрЃў

| рЃАрЃљрЃ«рЃћрЃџрЃў | рЃбрЃўрЃърЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
| --- | --- | --- |
| `rows_and_cols` | `STRING` | рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџ рЃфрЃ«рЃарЃўрЃџрЃерЃў. рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџ рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃерЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў рЃўрЃгрЃДрЃћрЃЉрЃљ 1-рЃўрЃЌ, рЃАрЃЋрЃћрЃбрЃћрЃЉрЃў рЃўрЃгрЃДрЃћрЃЉрЃљ A-рЃЊрЃљрЃю, рЃ«рЃЮрЃџрЃЮ OutputLists 0-рЃЊрЃљрЃю рЃўрЃгрЃДрЃћрЃЉрЃљ (рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃЊрЃарЃЮрЃА `select-nth`). |
| `header_rows` | `INT` | рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў x рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃАрЃўрЃўрЃЊрЃљрЃю. рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃЌрЃБ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЌ рЃАрЃЋрЃћрЃбрЃў `rows_and_cols`-рЃерЃў. |
| `header_cols` | `INT` | рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЌ рЃърЃўрЃарЃЋрЃћрЃџрЃў x рЃАрЃЋрЃћрЃбрЃў рЃАрЃўрЃўрЃЊрЃљрЃю. рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃЌрЃБ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЌ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў `rows_and_cols`-рЃерЃў. |
| `select_nth` | `INT` | рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃЮрЃюрЃўрЃерЃюрЃћрЃЌ nth рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў (0-рЃЊрЃљрЃю рЃўрЃгрЃДрЃћрЃЉрЃљ). рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮрЃљ `PrimitiveInt+control_after_generate=increment` рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў. |
| `string_or_base64` | `STRING` | CSV/TSV рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃљрЃю рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃў рЃфрЃ«рЃарЃўрЃџрЃў рЃЉрЃљрЃќрЃўрЃА-64 рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃЌ (`.ods .xlsx .xls`-рЃўрЃАрЃЌрЃЋрЃўрЃА). рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `Load Any File` рЃюрЃЮрЃЊрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃЉрЃљрЃќрЃўрЃА-64 рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃЌ рЃЕрЃљрЃАрЃљрЃбрЃЋрЃўрЃарЃЌрЃљрЃЊ. |
