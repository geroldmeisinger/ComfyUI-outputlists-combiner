## XYZ-GridPlot

![XYZ-GridPlot](XyzGridPlot/XyzGridPlot.png)

(包含 ComfyUI workflow)

從一組圖片列表生成 XYZ 網格圖。
它接受一個圖片列表（包括批次），並首先將其展平為一個長列表（因此 `batch_size=1`）。

**網格形狀**
透過以下方式決定網格的形狀：
1. 行標籤的數量
2. 列標籤的數量
3. 剩餘的子圖片。
您可以使用 `order=inside_out` 反轉圖片選擇順序（當 `batch_size>1` 且您希望標記批次時很有用）。

**對齊方式**
* 如果標籤被換行到下一行，整個軸都被視為「多行」，並在頂部對齊，使用兩端對齊的間距。
* 如果所有標籤都是數字或都以數字結尾（例如 `strength: 1.`），整個軸都被視為「數值」，並向右對齊。
* 所有其他文字被視為「單行」，並居中對齊。
* 列的單行和數值標籤在底部對齊，而行的標籤則在中間垂直對齊。

**字體大小**
* 列標籤區域的高度由 `font_size` 或「任何行中最大子圖片打包高度的一半」（取其較大者）決定。
* 行標籤區域的寬度由子圖片打包中最寬的寬度決定（最小為 256px）。
* 文字會被縮小直到適合為止（最小到 `font_size_min=6`），並對整個軸（行標籤或列標籤）使用相同的字體大小。
如果字體大小已達最小值，則截斷剩餘的文字。

**子圖片打包**
將子圖片（通常來自批次）塑形成最方形的區域（「子圖片打包」），除非 `output_is_list=True`，這種情況下每個單元格只使用一張圖片，並創建一個整體圖片網格列表。
您可以使用這個圖片網格列表連接到另一個 XyzGridPlot 節點來創建超級網格。
如果子圖片由不同大小的批次組成，則用空白圖片填補缺失的單元格。
每個單元格中的圖片數量（包括批次圖片）必須是 `rows * columns` 的倍數。

### 輸入

| 名稱 | 類型 | 描述 |
| --- | --- | --- |
| `images` | `IMAGE` | 一組圖片列表（包括批次） |
| `row_labels` | `*` | 左側的行標籤文字 |
| `col_labels` | `*` | 上方的列標籤文字 |
| `gap` | `INT` | 子圖片打包之間的間距。請注意子圖片內部不使用間距。如果您希望子圖片之間有間距，請連接另一個 XyzGridPlot 節點。 |
| `font_size` | `FLOAT` | 目標字體大小。文字會被縮小直到適合為止（最小到 `font_size_min=6`）。 |
| `row_label_orientation` | `COMBO` | 行標籤的文字方向。如果您想節省空間，這會很有用。 |
| `order` | `BOOLEAN` | 定義圖片處理順序。這僅在您有子圖片時相關。如果您有 `batch_size>1` 且想繪製批次時很有用。 |
| `output_is_list` | `BOOLEAN` | 僅當您有子圖片或想創建超級網格時才相關。 |

### 輸出

| 名稱 | 類型 | 描述 |
| --- | --- | --- |
| `image` | `IMAGE 𝌠` | XYZ-GridPlot 圖片。如果 `output_is_list=True`，則創建一個圖片列表，您可以連接到另一個 XYZ-GridPlot 節點來創建超級網格。 |
