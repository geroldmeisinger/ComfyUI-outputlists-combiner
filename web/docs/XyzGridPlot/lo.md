## XYZ-GridPlot

![XYZ-GridPlot](XyzGridPlot/XyzGridPlot.png)

(ComfyUI workflow ໄດ້ມີ)

ສ້າງ XYZ-Gridplot ຈາກລາຍຊື່ຮູບພາບ.
ມັນຮັບລາຍຊື່ຮູບພາບ (bao gồm batches) ແລະ ສ້າງເປັນລາຍຊື່ຍາວ (ດັ່ງນັ້ນ `batch_size=1`).

**ຮູບແບບຂອງເຄື່ອງໜັງສື**
ກຳນົດຮູບແບບຂອງເຄື່ອງໜັງສືໂດຍ:
1. ຈຳນວນຂະໜາດຂອງແຖວທີ່ມີຕົວເລື່ອນ
2. ຈຳນວນຂະໜາດຂອງແຖວທີ່ມີຕົວເລື່ອນ
3. ຮູບພາບເວັ້ນທີ່ຍັງເຫັນຢູ່.
ທ່ານສາມາດໃຊ້ `order=inside_out` ເພື່ອປ່ຽນການເລືອກຮູບພາບ (ເປັນເຄິ່ງທີ່ເປັນເຄິ່ງສຳຄັນຖ້າ `batch_size>1` ແລະ ທ່ານຕ້ອງການຕົວເລື່ອນ batch).

**ການຈັດລຽນ**
* ຖ້າຕົວເລື່ອນຖືກຈັດໃນແຖວຕໍ່ໄປ ຍາດໜຶ່ງຂອງແຖວຈະຖືກພິຈາລະນາວ່າ "ຫຼາຍແຖວ" ແລະ ຈັດລຽນໃນເທິງຂວາດດ້ວຍການຈັດລຽນຂະໜາດທີ່ເທົ່າກັນ.
* ຖ້າຕົວເລື່ອນທັງໝົດເປັນຕົວເລກ ຫຼື ພາຍຫຼັງຂອງຕົວເລື່ອນເປັນຕົວເລກ (ເຊັ່ນ `strength: 1.`) ຍາດໜຶ່ງຂອງແຖວຈະຖືກພິຈາລະນາວ່າ "ຕົວເລື່ອນຕົວເລກ" ແລະ ຈັດລຽນໃນຂວາດຂວາດ.
* ທຸກຢ່າງອື່ນໆຖືກພິຈາລະນາວ່າ "ແຖວດຽວ" ແລະ ຈັດລຽນໃນກາງ.
* ຈັດລຽນຕົວເລື່ອນແຖວດຽວ ແລະ ຕົວເລື່ອນຕົວເລກໃນແຖວຂວາດຂວາດ, ແລະ ຈັດລຽນໃນກາງຂອງແຖວ.

**ຂະໜາດຂອງຕົວອັກສອນ**
* ຊໍານ້ຳຂອງເຂດຕົວເລື່ອນແຖວແມ່ນກຳນົດໂດຍ `font_size` ຫຼື `ນິ້ວຂອງຮູບພາບເວັ້ນໃນແຖວໃດໃດ` (ໃນຂະນະທີ່ໃດໃດ).
* ຊໍານ້ຳຂອງເຂດຕົວເລື່ອນແຖວແມ່ນກຳນົດໂດຍຊໍານ້ຳຂອງຮູບພາບເວັ້ນທີ່ຍັງເຫັນຢູ່ (ມີຄ່າຕໍ່ຕ້ອງຂັ້ນຕໍ່ 256px).
* ທຸກຕົວອັກສອນຈະຖືກຫຼັ້ມເຖິງການຈັດລຽນໃນຂະນະທີ່ໃດໃດ (ເຖິງ `font_size_min=6`) ແລະ ຈະໃຊ້ຂະໜາດຂອງຕົວອັກສອນເທົ່າກັນໃນແຖວທັງໝົດ (ຕົວເລື່ອນແຖວຫຼືຕົວເລື່ອນແຖວຂວາດ).
ຖ້າຂະໜາດຂອງຕົວອັກສອນແມ່ນຕົວເລື່ອນຂັ້ນຕໍ່ແລ້ວ, ຈະຕັດຕົວອັກສອນທີ່ຍັງເຫັນຢູ່.

**ການຈັດລຽນຮູບພາບເວັ້ນ**
ຈັດຮູບພາບເວັ້ນ (ສຳເລັດຈາກ batch) ເປັນເຂດທີ່ສຳເລັດທີ່ສຸດ (ເປັນ "ການຈັດລຽນຮູບພາບເວັ້ນ"), ບໍ່ໃຊ້ `output_is_list=True`, ຈະໃຊ້ຮູບພາບດຽວໃນແຕ່ລະແຖວແລະສ້າງລາຍຊື່ຂອງເຂດຮູບພາບທີ່ສຳເລັດທີ່ສຸດ.
ທ່ານສາມາດໃຊ້ລາຍຊື່ຂອງເຂດຮູບພາບນີ້ເພື່ອເຊື່ອມຕໍ່ເຂດ XyzGridPlot ເລື່ອນອີກເພື່ອສ້າງເຂດທີ່ໃຫ້ສຳເລັດທີ່ສຸດ.
ຖ້າຮູບພາບເວັ້ນມີ batch ທີ່ມີຂະໜາດຕົວເລື່ອນຕໍ່ຕ້ອງ, ຈະຕັ້ງຮູບພາບເວັ້ນທີ່ບໍ່ມີໃນແຖວທີ່ຕ້ອງການໃສ່ໃນເຂດທີ່ບໍ່ມີຂະໜາດ.
ຈຳນວນຂອງຮູບພາບໃນແຖວ (ທີ່ເປັນຮູບພາບ batch) ຈະຕ້ອງເປັນເລກສຳເລັດຂອງ `rows * columns`.

### ການເຂົ້າຂໍ້ມູນ

| ຊື່ | ພະລົມ | ການອະທິບາຍ |
| --- | --- | --- |
| `images` | `IMAGE` | ລາຍຊື່ຮູບພາບ (bao gồm batches) |
| `row_labels` | `*` | ຕົວເລື່ອນແຖວຂວາດຂ້າຢູ່ຂ້າງຊ້າຍ |
| `col_labels` | `*` | ຕົວເລື່ອນແຖວຂວາດຂ້າຢູ່ຂ້າງເທິງ |
| `gap` | `INT` | ການຫຼັ້ມຂອງການຈັດລຽນຮູບພາບເວັ້ນ. ຄຳເຕືອນວ່າໃນຕົວເລື່ອນເອງບໍ່ໃຊ້ການຫຼັ້ມຂອງການຈັດລຽນຮູບພາບເວັ້ນ. ຖ້າທ່ານຕ້ອງການການຫຼັ້ມຂອງການຈັດລຽນຮູບພາບເວັ້ນ ເຊື່ອມຕໍ່ເຂດ XyzGridPlot ເລື່ອນອີກ. |
| `font_size` | `FLOAT` | ຂະໜາດຂອງຕົວອັກສອນທີ່ຕ້ອງການ. ທຸກຕົວອັກສອນຈະຖືກຫຼັ້ມເຖິງການຈັດລຽນໃນຂະນະທີ່ໃດໃດ (ເຖິງ `font_size_min=6`). |
| `row_label_orientation` | `COMBO` | ການຈັດລຽນຂອງຕົວເລື່ອນແຖວຂວາດຂ້າຢູ່ຂ້າງຊ້າຍ. ເປັນເຄິ່ງທີ່ເປັນເຄິ່ງສຳຄັນຖ້າທ່ານຕ້ອງການປະຢັດສະພາບ. |
| `order` | `BOOLEAN` | ກຳນົດໃນການຈັດລຽນຂອງຮູບພາບໃນຂະນະທີ່ໃດໃດ. ນີ້ສຳຄັນຖ້າທ່ານມີຮູບພາບເວັ້ນ. ເປັນເຄິ່ງທີ່ເປັນເຄິ່ງສຳຄັນຖ້າ `batch_size>1` ແລະ ທ່ານຕ້ອງການສ້າງ batch. |
| `output_is_list` | `BOOLEAN` | ນີ້ສຳຄັນຖ້າທ່ານມີຮູບພາບເວັ້ນ ຫຼື ທ່ານຕ້ອງການສ້າງເຂດທີ່ໃຫ້ສຳເລັດທີ່ສຸດ. |
