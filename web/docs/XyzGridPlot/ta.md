## XYZ-GridPlot

![XYZ-GridPlot](XyzGridPlot/XyzGridPlot.png)

(ComfyUI workflow இடப்பட்டுள்ளது)

படங்களின் பட்டியலிலிருந்து XYZ-Gridplot ஐ உருவாக்குகிறது.
இது படங்களின் பட்டியலை (பேட்ச்களை உள்ளடக்கி) முதலில் நீளமான பட்டியலாக மாற்றுகிறது (இதனால் `batch_size=1`).

**கட்டமைப்பு வடிவம்**
கட்டமைப்பின் வடிவத்தை கீழ்கண்டவற்றால் குறிப்பிடுகிறது:
1. வரிசை லேபிள்களின் எண்ணிக்கை
2. நெடுவரிசை லேபிள்களின் எண்ணிக்கை
3. மீதமுள்ள துணை-படங்கள்.
`order=inside_out` ஐப் பயன்படுத்தி படங்களை தலைகீழாக தேர்ந்தெடுக்க முடியும் (இது `batch_size>1` என்றால் பயனுள்ளது மற்றும் பேட்ச்களை லேபிள் செய்ய விரும்பினால்).

**சீரமைப்பு**
* ஒரு லேபிள் அடுத்த வரிசைக்கு மாற்றப்பட்டால் முழு அச்சும் "மல்டிலைன்" என கருதப்பட்டு, அவற்றை மேல் முறையில் சீரமைத்து சம இடைவெளியில் வைக்கப்படும்.
* லேபிள்கள் அனைத்தும் எண்களாக இருந்தால் அல்லது எண்களாக முடியும் (எ.கா: `strength: 1.`) அவை முழு அச்சும் "எண்முறை" என கருதப்பட்டு வலது பக்கம் சீரமைக்கப்படும்.
* மற்ற உரைகள் "சிறிய வரிசை" என கருதப்பட்டு மையமாக சீரமைக்கப்படும்.
* சிறிய வரிசை மற்றும் எண்முறை லேபிள்களை நெடுவரிசைகளுக்கு கீழ் சீரமைக்கப்படும், வரிசைகளுக்கு நடுவே சீரமைக்கப்படும்.

**எழுத்துரு அளவு**
* நெடுவரிசை லேபிள் பகுதியின் உயரம் `font_size` அல்லது ஒரு வரிசையில் மிகப்பெரிய துணை-படங்களின் உயரத்தின் அரை மதிப்பால் குறிப்பிடப்படும் (அது கூடுதலாக இருந்தால்).
* வரிசை லேபிள் பகுதியின் அகலம் துணை-படங்களின் மிகப்பெரிய அகலத்தால் குறிப்பிடப்படும் (குறைந்தபட்சம் 256px).
* உரை பொருந்தும்படி சுருக்கப்படும் (மிகக்குறைந்த அளவு `font_size_min=6` வரை) மற்றும் முழு அச்சில் ஒரே எழுத்துரு அளவு பயன்படுத்தப்படும் (வரிசை லேபிள்கள் அல்லது நெடுவரிசை லேபிள்கள்).
எழுத்துரு அளவு ஏற்கனவே குறைந்த அளவில் இருந்தால், மீதமுள்ள உரைகளை வெட்டுகிறது.

**துணை-படங்கள் கலைப்பு**
துணை-படங்களை (பேட்ச்களிலிருந்து பெறப்பட்டவை) மிகச்சது சதுரமான பகுதிக்கு மாற்றுகிறது ("துணை-படங்கள் கலைப்பு"), `output_is_list=True` என்றால் ஒரு செல்லுக்கு ஒரு படம் மட்டுமே பயன்படுத்தப்படுகிறது மற்றும் முழு பட கட்டமைப்புகளின் பட்டியலை உருவாக்குகிறது.
இந்த பட கட்டமைப்புகளின் பட்டியலை மற்றொரு XyzGridPlot நோட்டுடன் இணைக்க முடியும் மற்றும் சூப்பர்-கட்டமைப்புகளை உருவாக்க முடியும்.
துணை-படங்கள் வெவ்வேறு அளவு பேட்ச்களாக இருந்தால், காலியான செல்களை நிரப்புகிறது.
செல்லுக்கு உள்ள படங்களின் எண்ணிக்கை (பேட்ச்களாக இருக்கும் படங்கள் உள்ளது) `rows * columns` இன் மடங்காக இருக்க வேண்டும்.

### உள்ளீடுகள்

| பெயர் | வகை | விளக்கம் |
| --- | --- | --- |
| `images` | `IMAGE` | படங்களின் பட்டியல் (பேட்ச்களை உள்ளடக்கி) |
| `row_labels` | `*` | இடப்பக்கத்தில் உள்ள வரிசை லேபிள்கள் |
| `col_labels` | `*` | மேல்பக்கத்தில் உள்ள நெடுவரிசை லேபிள்கள் |
| `gap` | `INT` | துணை-படங்களின் இடைவெளி. துணை-படங்களுக்கு இடையே இடைவெளி இல்லை என்பதை குறிப்பிடுகிறது. துணை-படங்களுக்கு இடையே இடைவெளி வேண்டினால் மற்றொரு XyzGridPlot நோட்டுடன் இணைக்கவும். |
| `font_size` | `FLOAT` | இலக்கு எழுத்துரு அளவு. உரை பொருந்தும்படி சுருக்கப்படும் (மிகக்குறைந்த அளவு `font_size_min=6` வரை). |
| `row_label_orientation` | `COMBO` | வரிசை லேபிள்களின் உரை திசை. இடம் சேமிக்க விரும்பினால் பயனுள்ளது. |
| `order` | `BOOLEAN` | படங்களை செயல்முறையில் செயல்படுத்த வேண்டிய வரிசையை வரையறுக்கிறது. இது துணை-படங்கள் இருந்தால் மட்டுமே பயனுள்ளது. `batch_size>1` என்றால் பேட்ச்களை வரையறுக்க விரும்பினால் பயனுள்ளது. |
| `output_is_list` | `BOOLEAN` | இது துணை-படங்கள் இருந்தால் மட்டுமே பயனுள்ளது அல்லது முழு பட கட்டமைப்புகளின் பட்டியலை உருவாக்க வேண்டிய வழியை வரையறுக்கிறது. |
