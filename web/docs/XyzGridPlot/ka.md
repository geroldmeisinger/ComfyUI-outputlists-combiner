## XYZ-GridPlot

![XYZ-GridPlot](XyzGridPlot/XyzGridPlot.png)

(ComfyUI рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ)

рЃљрЃЏ рЃюрЃЮрЃЊрЃўрЃА рЃЏрЃўрЃћрЃа рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ XYZ-Gridplot рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃўрЃЊрЃљрЃю.
рЃўрЃА рЃўрЃдрЃћрЃЉрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃА (рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃА) рЃЊрЃљ рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃњрЃљрЃЊрЃЏрЃЮрЃљрЃЦрЃфрЃћрЃЋрЃА рЃњрЃарЃФрЃћрЃџ рЃАрЃўрЃљрЃерЃў (рЃљрЃЏрЃўрЃбрЃЮрЃЏ `batch_size=1`).

**рЃЉрЃџрЃЮрЃЎрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ**
рЃЉрЃџрЃЮрЃЎрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ:
1. рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃГрЃЊрЃћрЃћрЃЉрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃў
2. рЃАрЃЋрЃћрЃбрЃўрЃА рЃГрЃЊрЃћрЃћрЃЉрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃў
3. рЃЊрЃљрЃарЃЕрЃћрЃюрЃўрЃџрЃў рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў.
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ `order=inside_out` рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃерЃЋрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ (рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃўрЃљ `batch_size>1`-рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃА рЃњрЃАрЃБрЃарЃЌ рЃГрЃЊрЃћрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ).

**рЃАрЃгрЃЮрЃарЃћрЃЉрЃљ**
* рЃЌрЃБ рЃГрЃЊрЃћ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃ«рЃљрЃќрЃерЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА, рЃЏрЃЌрЃћрЃџрЃў рЃдрЃћрЃарЃФрЃў рЃљрЃарЃўрЃА "рЃЏрЃарЃљрЃЋрЃљрЃџрЃ«рЃљрЃќрЃўрЃљрЃюрЃў" рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃќрЃћрЃЊрЃљ рЃЏрЃ«рЃљрЃарЃћрЃА, рЃњрЃљрЃюрЃљрЃЌрЃћрЃАрЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ.
* рЃЌрЃБ рЃДрЃЋрЃћрЃџрЃљ рЃГрЃЊрЃћ рЃљрЃарЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃў рЃљрЃю рЃДрЃЋрЃћрЃџрЃљ рЃЉрЃЮрЃџрЃЮ рЃЏрЃўрЃЏрЃЊрЃћрЃЉрЃљрЃарЃћрЃЉрЃА рЃарЃўрЃфрЃ«рЃЋрЃќрЃћ (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ `strength: 1.`) рЃЏрЃЌрЃћрЃџрЃў рЃдрЃћрЃарЃФрЃў рЃљрЃарЃўрЃА "рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў" рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ.
* рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃбрЃћрЃЦрЃАрЃбрЃў рЃљрЃарЃўрЃА "рЃћрЃарЃЌрЃ«рЃљрЃќрЃўрЃљрЃюрЃў" рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃерЃБрЃљрЃерЃў.
* рЃћрЃарЃЌрЃ«рЃљрЃќрЃўрЃљрЃюрЃў рЃЊрЃљ рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃГрЃЊрЃћрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃарЃћрЃЉрЃљ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃљрЃарЃўрЃА рЃЦрЃЋрЃћрЃЊрЃљ рЃЏрЃ«рЃљрЃарЃћрЃА, рЃ«рЃЮрЃџрЃЮ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃљрЃарЃўрЃА рЃерЃБрЃљрЃерЃў рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ.

**рЃерЃарЃўрЃцрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ**
* рЃАрЃЋрЃћрЃбрЃўрЃА рЃГрЃЊрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃБрЃџрЃўрЃљ `font_size`-рЃўрЃЌ рЃљрЃю `рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА рЃюрЃљрЃ«рЃћрЃЋрЃљрЃарЃў` (рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃћрЃбрЃўрЃљ).
* рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃГрЃЊрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃћ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃАрЃўрЃњрЃљрЃюрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў 256px).
* рЃбрЃћрЃЦрЃАрЃбрЃў рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃюрЃљрЃЏ, рЃАрЃљрЃюрЃљрЃЏ рЃљрЃа рЃЕрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ (рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ `font_size_min=6`) рЃЊрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃерЃарЃўрЃцрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљрЃА рЃЏрЃЌрЃћрЃџрЃў рЃдрЃћрЃарЃФрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃГрЃЊрЃћрЃћрЃЉрЃў рЃљрЃю рЃАрЃЋрЃћрЃбрЃўрЃА рЃГрЃЊрЃћрЃћрЃЉрЃў).
рЃЌрЃБ рЃерЃарЃўрЃцрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃБрЃЎрЃЋрЃћ рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃўрЃљ, рЃЏрЃЮрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃў рЃљрЃЏрЃЮрЃГрЃарЃўрЃџрЃўрЃљ.

**рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ**
рЃЏрЃЮрЃўрЃ«рЃАрЃћрЃюрЃўрЃћрЃЌ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў (рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃў) рЃБрЃЏрЃћрЃбрЃћрЃАрЃљрЃЊ рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ (РђърЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљРђю), рЃњрЃљрЃарЃЊрЃљ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃАрЃљ, рЃЌрЃБ `output_is_list=True`, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃДрЃЮрЃЋрЃћрЃџрЃў рЃБрЃ»рЃарЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЦрЃЏрЃюрЃўрЃА рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃА рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃћрЃЮ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ.
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃљрЃЏ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃћрЃЮ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ, рЃарЃЮрЃЏ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃћрЃЮ рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃў.
рЃЌрЃБ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃќрЃЮрЃЏрЃўрЃАрЃљрЃљ, рЃЏрЃЮрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃБрЃ»рЃарЃћрЃЉрЃў рЃерЃћрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃЌ.
рЃБрЃ»рЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃў (рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ) рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА `rows * columns`-рЃўрЃА рЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃў.

| `images` | `IMAGE` | рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ (рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ) |
| `row_labels` | `*` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃГрЃЊрЃћрЃћрЃЉрЃў рЃЏрЃљрЃарЃфрЃ«рЃћрЃюрЃљ рЃЏрЃ«рЃљрЃарЃћрЃА |
| `col_labels` | `*` | рЃАрЃЋрЃћрЃбрЃўрЃА рЃГрЃЊрЃћрЃћрЃЉрЃў рЃќрЃћрЃЊрЃљ рЃЏрЃ«рЃљрЃарЃћрЃА |
| `gap` | `INT` | рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃерЃЮрЃарЃўрЃА рЃАрЃўрЃЏрЃљрЃарЃЌрЃџрЃћ. рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃАрЃўрЃЏрЃљрЃарЃЌрЃџрЃћ рЃљрЃа рЃљрЃарЃўрЃА. рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ рЃАрЃўрЃЏрЃљрЃарЃЌрЃџрЃћ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА, рЃЊрЃљрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЌ рЃАрЃ«рЃЋрЃљ XyzGridPlot рЃюрЃЮрЃЊрЃў. |
| `font_size` | `FLOAT` | рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃерЃарЃўрЃцрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ. рЃбрЃћрЃЦрЃАрЃбрЃў рЃерЃћрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃюрЃљрЃЏ, рЃАрЃљрЃюрЃљрЃЏ рЃљрЃа рЃЕрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ (рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ `font_size_min=6`). |
| `row_label_orientation` | `COMBO` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃГрЃЊрЃўрЃА рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљ. рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃўрЃљ, рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃЊрЃљрЃќрЃЮрЃњрЃЋрЃљ. |
| `order` | `BOOLEAN` | рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃўрЃЏ рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ. рЃћрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў. рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃўрЃљ `batch_size>1`-рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА рЃњрЃАрЃБрЃарЃЌ. |
| `output_is_list` | `BOOLEAN` | рЃћрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃЌ рЃЦрЃЋрЃћ-рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃљрЃю рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃћрЃЮ рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃњрЃАрЃБрЃарЃЌ. |
