## XYZ-GridPlot

![XYZ-GridPlot](XyzGridPlot/XyzGridPlot.png)

(Συμπεριλαμβάνεται ComfyUI workflow)

Δημιουργεί ένα XYZ-Gridplot από μια λίστα εικόνων.
Λαμβάνει μια λίστα εικόνων (συμπεριλαμβανομένων των batch) και τις επεξεργάζεται σε μια μεγάλη λίστα (έτσι `batch_size=1`).

**Σχήμα πλέγματος**
Καθορίζει το σχήμα του πλέγματος βάσει:
1. του αριθμού ετικετών γραμμών
2. του αριθμού ετικετών στηλών
3. των υπο-εικόνων που απομένουν.
Μπορείτε να χρησιμοποιήσετε `order=inside_out` για να αντιστρέψετε την επιλογή εικόνων (χρήσιμο αν `batch_size>1` και θέλετε να ετικετάρετε τα batch).

**Στοίχιση**
* Αν μια ετικέτα περικυματίζεται στην επόμενη γραμμή, ολόκληρος άξονας θεωρείται "πολλαπλής γραμμής" και στοιχίζεται στην κορυφή με διαστάσεις πλήρως διακεκριμένες.
* Αν όλες οι ετικέτες είναι αριθμοί ή όλες τελειώνουν σε αριθμούς (π.χ. `strength: 1.`) ολόκληρος άξονας θεωρείται "αριθμητικός" και στοιχίζεται στα δεξιά.
* Όλα τα άλλα κείμενα θεωρούνται "μονής γραμμής" και στοιχίζονται στο κέντρο.
* Στοιχίζει ετικέτες μονής γραμμής και αριθμητικές ετικέτες για στήλες στο κάτω μέρος, και για γραμμές στο κέντρο κατά μήκος.

**Μέγεθος γραμματοσειράς**
* Το ύψος της περιοχής ετικετών στήλης καθορίζεται από `font_size` ή `το μισό του μεγαλύτερου ύψους πακέτων υπο-εικόνων σε κάθε γραμμή` (όποτε είναι μεγαλύτερο).
* Το πλάτος της περιοχής ετικετών γραμμής καθορίζεται από το πιο μεγάλο πλάτος των πακέτων υπο-εικόνων (με ελάχιστο 256px).
* Το κείμενο συμπιέζεται μέχρι να χωρέσει (μέχρι `font_size_min=6`) και χρησιμοποιεί το ίδιο μέγεθος γραμματοσειράς για ολόκληρον άξονα (ετικέτες γραμμής ή ετικέτες στήλης).
Αν το μέγεθος γραμματοσειράς είναι ήδη στο ελάχιστο, κόβει τυχόν υπολειμματικό κείμενο.

**Πακέτωση υπο-εικόνων**
Σχηματίζει τις υπο-εικόνες (συνήθως από batch) στην πιο τετράγωνη περιοχή (την "πακέτωση υπο-εικόνων"), εκτός αν `output_is_list=True`, οπότε χρησιμοποιεί μόνο μια εικόνα για κάθε κελί και δημιουργεί μια λίστα από ολόκληρα πλέγματα εικόνων.
Μπορείτε να χρησιμοποιήσετε αυτή τη λίστα πλεγμάτων εικόνων για να συνδέσετε έναν άλλο κόμβο XyzGridPlot για να δημιουργήσετε υπερ-πλέγματα.
Αν οι υπο-εικόνες αποτελούνται από batch διαφορετικών μεγεθών, συμπληρώνει τα λείποντα κελιά με κενές εικόνες.
Ο αριθμός των εικόνων ανά κελί (συμπεριλαμβανομένων των batch εικόνων) πρέπει να είναι πολλαπλάσιο του `rows * columns`.

### Εισόδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `images` | `IMAGE` | Μια λίστα εικόνων (συμπεριλαμβανομένων των batch) |
| `row_labels` | `*` | Κείμενα ετικετών γραμμής στην αριστερή πλευρά |
| `col_labels` | `*` | Κείμενα ετικετών στήλης στην κορυφή |
| `gap` | `INT` | Απόσταση μεταξύ των πακέτων υπο-εικόνων. Σημειώστε ότι μέσα στις υπο-εικόνες δεν χρησιμοποιείται απόσταση. Αν θέλετε απόσταση μεταξύ των υπο-εικόνων συνδέστε έναν άλλο κόμβο XyzGridPlot. |
| `font_size` | `FLOAT` | Στόχος μεγέθους γραμματοσειράς. Το κείμενο θα συμπιέζεται μέχρι να χωρέσει (μέχρι `font_size_min=6`). |
| `row_label_orientation` | `COMBO` | Προσανατολισμός κειμένου των ετικετών γραμμής. Χρήσιμο αν θέλετε να εξοικονομήσετε χώρο. |
| `order` | `BOOLEAN` | Καθορίζει ποια σειρά θα επεξεργαστούν οι εικόνες. Αυτό είναι σημαντικό μόνο αν έχετε υπο-εικόνες. Χρήσιμο αν `batch_size>1` και θέλετε να σχεδιάσετε τα batch. |
| `output_is_list` | `BOOLEAN` | Αυτό είναι σημαντικό μόνο αν έχετε υπο-εικόνες ή θέλετε να δημιουργήσετε υπερ-πλέγματα. |

### Έξοδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `image` | `IMAGE 𝌠` | Η εικόνα XYZ-GridPlot. Αν `output_is_list=True` δημιουργεί μια λίστα εικόνων που μπορείτε να συνδέσετε σε έναν άλλο κόμβο XYZ-GridPlot για να δημιουργήσετε υπερ-πλέγματα. |
