## XYZ-GridPlot

![XYZ-GridPlot](XyzGridPlot/XyzGridPlot.png)

(បានរួមបញ្ចូល ComfyUI workflow)

បង្កើត XYZ-Gridplot ពីបញ្ជីរូបភាព។
វាប្រើប្រាស់បញ្ជីរូបភាព (រួមបញ្ចូល batch) ហើយបញ្ចូលវាទៅជាបញ្ជីវែងជាមួយ (ដូច្នេះ `batch_size=1`)

**ទម្រង់ប្រអប់**
កំណត់ទម្រង់ប្រអប់ដោយ:
1. ចំនួនស្លាក​បន្ទាត់ផ្ទៃខាងក្រោម
2. ចំនួនស្លាក​បន្ទាត់ផ្ទៃខាងលិច
3. រូបភាពរបស់ផ្នែក​ដែលនៅខាងក្នុង។
អ្នកអាចប្រើ `order=inside_out` ដើម្បីបញ្ច្រាសការជ្រើសរើសរូបភាព (មានប្រយោជន៍ប្រសិនបើ `batch_size>1` ហើយអ្នកចង់ស្លាក batch)។

**ការតំរូវ**
* ប្រសិនបើស្លាក​ត្រូវបានបំបែកទៅជារបារប់ទេ ប្រអប់ទាំងអស់ត្រូវបានគ្រប់គ្រងដោយ "multiline" ហើយតំរូវវាទៅខាងលើដោយការចែកចាយដោយមានស្រាវិត។
* ប្រសិនបើស្លាកទាំងអស់គឺជាលេខ ឬទាំងអស់បញ្ចប់ដោយលេខ (ឧទាហរណ៍ `strength: 1.`) ប្រអប់ទាំងអស់ត្រូវបានគ្រប់គ្រងដោយ "numeric" ហើយតំរូវវាទៅខាងស្តាំ។
* អត្ថបទផ្សេងទៀតគឺត្រូវបានគ្រប់គ្រងដោយ "singleline" ហើយតំរូវវាទៅកណ្តាល។
* តំរូវស្លាក singleline និង numeric សម្រាប់ជួរឈរក្នុងការតំរូវទៅខាងក្រោម ហើយសម្រាប់ជួរឈរក្នុងការតំរូវទៅកណ្តាលក្នុងការផ្ទះ។

**ទំហំពុម្ពអក្សរ**
* កំពស់តំបន់ស្លាកជួរឈរត្រូវបានកំណត់ដោយ `font_size` ឬ `ការផ្ទះកំពស់របស់រូបភាពដែលមានទំហំធំជាងគេនៅក្នុងជួរឈរ` (ដោយមិនចាប់ពីគ្នារវាងពួកវាទេ)។
* ទទឹងតំបន់ស្លាកជួរឈរត្រូវបានកំណត់ដោយទទឹងដែលមានទំហំធំជាងគេនៃការផ្ទះរូបភាព (ដោយមានអំណាចចាប់ពី 256px)។
* អត្ថបទត្រូវបានបង្រួមដោយការបន្ថយរហូតដល់វាទំនាក់ទំនាក់ (ទៅកាន់ `font_size_min=6`) ហើយប្រើពុម្ពអក្សរដូចគ្នាទៅលើប្រអប់ទាំងអស់ (ស្លាកជួរឈរ ឬស្លាកជួរក្រោម)។
ប្រសិនបើទំហំពុម្ពអក្សរបានបង្រួមរួច វានឹងកាត់អត្ថបទដែលនៅខាងក្រោយ។

**ការផ្ទះរូបភាព**
កំណត់ទម្រង់រូបភាពរបស់ផ្នែករូបភាព (ធម្មតាត្រូវបានប្រើដោយ batch) ទៅក្នុងតំបន់ដែលមានទំហំតូចជាងគេ (ទំហំ "sub-images packing") លុះបែកពី `output_is_list=True` ដែលប្រើតែរូបភាពមួយដែលមានសម្រាប់ចំណុចនីមួយៗ ហើយបង្កើតបញ្ជីរបស់ប្រអប់រូបភាពទាំងមូល។
អ្នកអាចប្រើបញ្ជីរបស់ប្រអប់រូបភាពនេះដើម្បីភ្ជាប់នឹងកម្មវិធី XyzGridPlot ផ្សេងទៀតដើម្បីបង្កើត super-grids។
ប្រសិនបើរូបភាពរបស់ផ្នែករូបភាពមាន batch ដែលមានទំហំផ្សេងគ្នា វានឹងបំពេញចំណុចដែលខ្ថៃនៅក្នុងប្រអប់ដោយប្រើរូបភាពទទេះ។
ចំនួនរូបភាពត្រូវបានបញ្ចូលទៅក្នុងចំណុចនីមួយៗ (រួមបញ្ចូលរូបភាព batch) ត្រូវតែជាចំនួនដែលមានគុណនេះនៃ `rows * columns`។

### បញ្ចូល

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
| --- | --- | --- |
| `images` | `IMAGE` | បញ្ជីរូបភាព (រួមបញ្ចូល batch) |
| `row_labels` | `*` | អត្ថបទស្លាកជួរឈរនៅខាងឆ្វេង |
| `col_labels` | `*` | អត្ថបទស្លាកជួរក្រោមនៅខាងលើ |
| `gap` | `INT` | ចន្លោះរវាងការផ្ទះរូបភាព។ សូមចំណាំថាក្នុងខាងក្នុងរូបភាពរបស់ផ្នែករូបភាពនោះគ្មានចន្លោះ។ ប្រសិនបើអ្នកចង់មានចន្លោះរវាងរូបភាពរបស់ផ្នែករូបភាព ត្រូវភ្ជាប់កម្មវិធី XyzGridPlot ផ្សេងទៀត។ |
| `font_size` | `FLOAT` | ទំហំពុម្ពអក្សរដែលចង់។ អត្ថបទនឹងត្រូវបានបង្រួមដោយការបន្ថយរហូតដល់វាទំនាក់ទំនាក់ (ទៅកាន់ `font_size_min=6`)។ |
| `row_label_orientation` | `COMBO` | ទិសដៅអត្ថបទស្លាកជួរឈរ។ មានប្រយោជន៍ប្រសិនបើអ្នកចង់រក្សាទំហំ។ |
| `order` | `BOOLEAN` | កំណត់ការតំរូវរបស់រូបភាពដែលត្រូវបានដំណើរការ។ វាត្រូវបានគ្រប់គ្រងប្រសិនបើអ្នកមានផ្នែករូបភាព។ មានប្រយោជន៍ប្រសិនបើ `batch_size>1` ហើយអ្នកចង់បង្កើតប្រអប់ batch ។ |
| `output_is_list` | `BOOLEAN` | វាត្រូវបានគ្រប់គ្រងប្រសិនបើអ្នកមានផ្នែករូបភាព ឬអ្នកចង់បង្កើត super-grids។ |
