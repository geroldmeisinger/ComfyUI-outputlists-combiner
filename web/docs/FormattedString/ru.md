## Форматированная строка

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow включен)

Создает строку, которая содержит переменные-заполнители и заменяет их соответствующими значениями.
Внутри использует python `str.format()`, см. [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Вы можете использовать `{a:.2f}` для округления числа с плавающей точкой до 2 знаков после запятой.
* Вы можете использовать `{a:05d}` для добавления ведущих нулей до 5, чтобы соответствовать суффиксу имени файла comfys `ComfyUI_00001_.png`.
* Если вы хотите написать `{ }` внутри строк (например, для JSON), вам нужно удвоить их: `{{ }}`.

Также применяет синтаксис *поиска и замены (S&R)*, такой как `%date:yyyy-MM-dd hh:mm:ss%` и `%KSampler.seed%`.
Таким образом, вы также можете использовать его как `GET-node`.
Обратите внимание, что "поиск и замена" происходит в контексте Javascript и выполняется до выполнения узла.
