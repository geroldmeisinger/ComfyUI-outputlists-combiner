## Форматлаған һөрөм

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow килеше)

Төрөлөгән һөрөмҙәрҙә һайланған һөрөмҙәрҙән һөрөм булдырып, уларҙың тәғәйән ҡыйммәттәре менән алмаштыра.
Python `str.format()` ҡулланыла, [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) ҡарағыҙ.
* `{a:.2f}` ҡулланып, һөрөмҙән 2 уныҡ ҡалып һөрөм булдырырға мөмкин.
* `{a:05d}` ҡулланып, comfys файл атама суффиксы `ComfyUI_00001_.png` менән тәғәйән ҡыҙыҡ 5 нөлдән һөрөм булдырырға мөмкин.
* һөрөмҙә `{ }` ҡуйырға һыҙыҡ (мәғәнәлә JSON) һөрөмҙәрҙә ҡулланырға мөмкин, уларҙың икенсе ҡуйылышы: `{{ }}`.

*Search & replace (S&R) синтаксисы* кебек ҡыҙыҡ булдырырға мөмкин, мәғәнәлә `%date:yyyy-MM-dd hh:mm:ss%` һәм `%KSampler.seed%`.
Шуға ҡарап, уларҙы `GET-node` ҡыҙыҡ булдырырға мөмкин.
"Search & replace" JavaScript контекстында һәм нод исполнение эшләмәй тән эшләнә.
