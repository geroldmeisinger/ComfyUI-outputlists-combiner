## Форматований рядок

![Formatted String](FormattedString/FormattedString.png)

(Включено ComfyUI workflow)

Створює рядок, який містить змінні-заповнювачі та замінює їх відповідними значеннями.
Використовує внутрішньо python `str.format()`, див. [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Ви можете використовувати `{a:.2f}` для округлення float до 2 знаків після коми.
* Ви можете використовувати `{a:05d}` для додавання до 5 ведучих нулів для відповідності суфіксу імені файлу Comfy `ComfyUI_00001_.png`.
* Якщо ви хочете використовувати `{ }` у своїх рядках (наприклад, для JSON), вам потрібно подвоїти їх: `{{ }}`.

Також застосовує *синтаксис пошуку та заміни (S&R)*, наприклад `%date:yyyy-MM-dd hh:mm:ss%` та `%KSampler.seed%`.
Таким чином, ви також можете використовувати його як `GET-node`.
Зауважте, що "пошук та заміна" відбувається в контексті Javascript і виконується до виконання вузла.
