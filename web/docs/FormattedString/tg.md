## Строка бо формат

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow дар инҷуд)

Сана мавзӯъи сатри мавзӯъи дар он ҷо мавҷуд будани манзӯри арзишҳои мувофиқ.
Аз `str.format()` истифода мешавад, инҷо [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Шумо метавонед `{a:.2f}`-ро барои гузоштани 2 рақами асоси ҳар ду ҳолат истифода баред.
* Шумо метавонед `{a:05d}`-ро барои пад кардани 5 ноли асосӣ истифода баред, то барои мувофиқат бо сурати номи файлҳои Comfy `ComfyUI_00001_.png`.
* Агар шумо хоҳед `{ }`-ро дар сатрҳои худ (масалан, барои JSON) гузоред, шумо бояд онҳоро дубора гузоред: `{{ }}`.

Ҳамчунин аз синтаксиси *ҷустуҷӯ ва иваз (S&R)* истифода мешавад, мисол: `%date:yyyy-MM-dd hh:mm:ss%` ва `%KSampler.seed%`.
Барои ин ҷо шумо метавонед онро ҳамчун `GET-node` истифода баред.
Диққат кунед, ки "ҷустуҷӯ ва иваз" дар муҳити JavaScript иҷро мешавад ва пеш аз иҷрои ҷузъ иҷро мешавад.
