## Пішімделген жазу

![Пішімделген жазу](FormattedString/FormattedString.png)

(ComfyUI жұмыс жоспары кірістірілген)

Жазуға орналастырылған айнымалыларды қамтиды және олардың сәйкес мәндерімен алмастырады.
Ішкі жұмыста python `str.format()` қолданылады, қараңыз [Python - Пішімделген жазу синтаксисі](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Бөлшек санды 2 дәрежелі түріне дейін дөңгелектеу үшін `{a:.2f}` қолдануға болады.
* Comfys файл атауының `ComfyUI_00001_.png` тұрғындағы тұрғын аяқталуына сәйкес 5 бастапқы нөлдерді қосу үшін `{a:05d}` қолдануға болады.
* Егер сіз жазуларыңызда `{ }` жазу керек болса (мысалы JSON үшін), оларды екі есе жазу керек: `{{ }}`.

Сондай-ақ *іздеу және ауыстыру (S&R) синтаксисін* қолданады, мысалы `%date:yyyy-MM-dd hh:mm:ss%` және `%KSampler.seed%`.
Сондықтан оны `GET-нод` ретінде де қолдануға болады.
Ескерту: "іздеу және ауыстыру" жұмысы Javascript контекстінде жүзеге асырылып, нод жұмысынан бұрын орындалады.
