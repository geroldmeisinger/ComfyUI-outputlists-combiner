## ខ្លឹមសារដែលបានរចនាប័ទ្ម

![Formatted String](FormattedString/FormattedString.png)

(បានរួមបញ្ចូល ComfyUI workflow)

បង្កើតខ្លឹមសារដែលមានអថេរកន្លែងទំនាក់ទំនាក់ ហើយជំនួសវាជាមួយនឹងតម្លៃដែលពាក់ព័ន្ធ។
ប្រើ `str.format()` របស់ Python ខាងក្នុង មើល [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) ។
* អ្នកអាចប្រើ `{a:.2f}` ដើម្បីបញ្ចប់លេខទស្សនាទៅជាចំនួន 2 ខ្ទង់។
* អ្នកអាចប្រើ `{a:05d}` ដើម្បីបន្ថែមលេខសូន្យ 5 ខាងមុខដើម្បីសមត្ថភាពជាមួយនឹងក្រុមហ៊ុន ComfyUI ដែលមានប្រភេទឈ្មោះឯកសារ `ComfyUI_00001_.png`។
* ប្រសិនបើអ្នកចង់សរសេរ `{ }` នៅក្នុងខ្លឹមសាររបស់អ្នក (ឧទាហរណ៍សម្រាប់ JSONs) អ្នកត្រូវប្រើពីរដង: `{{ }}`។

ក៏អនុវត្ត *សំណុំសំណុំសំណុំ (S&R)* ដូចជា `%date:yyyy-MM-dd hh:mm:ss%` និង `%KSampler.seed%`។
ដូច្នេះអ្នកក៏អាចប្រើវាជា `GET-node` បាន។
ចំណាំថាតែ "ស្វែងរក និងជំនួស" កើតឡើងនៅក្នុងបរិយាកាស Javascript ហើយដំណើរការមុនពេលដំណើរការនៃ node ។
