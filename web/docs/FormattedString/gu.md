## ફૉર્મેટેડ સ્ટ્રિંગ

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow સામેલ છે)

સ્ટ્રિંગ બનાવે છે જે પ્લેસહોલ્ડર વેરિએબલ્સ ધરાવે છે અને તેમના મૂલ્યો સાથે તેમની જગ્યા લેવાય છે.
પાયથૉન `str.format()` નો ઉપયોગ કરે છે, જો કે [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) જુઓ.
* તમે `{a:.2f}` નો ઉપયોગ કરી શકો છો તે ફ્લોટને 2 દશમલવ સ્થાનો સુધી લગભગ કરવા માટે.
* તમે `{a:05d}` નો ઉપયોગ કરી શકો છો તે પ્રારંભિક શૂન્યોને સારી રીતે પૂર્ણ કરવા માટે જે કોમ્ફીસ ફાઇલના પ્રત્યક્ષાની સંખ્યા સાથે મેળવાય છે `ComfyUI_00001_.png`.
* જો તમે તમારી સ્ટ્રિંગમાં `{ }` લખવા માંગતા હોય (ઉદાહરણ તરીકે JSONs માટે) તો તમારે તેને ડબલ કરવું પડશે: `{{ }}`.

*સર્ચ અને રીપ્લેસ (S&R) સિન્ટેક્સ* પણ લાગુ કરે છે જે જેવું છે `%date:yyyy-MM-dd hh:mm:ss%` અને `%KSampler.seed%`.
તેથી તમે તેને `GET-node` તરીકે પણ ઉપયોગ કરી શકો છો.
ધ્યાન આપો કે "સર્ચ અને રીપ્લેસ" જાવાસ્ક્રિપ્ટ સંદર્શનમાં થાય છે અને નોડ એક્સીક્યુશન પહેલાં ચલાવાય છે.
