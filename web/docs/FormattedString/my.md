## စာသားပြောင်းလဲမှု

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow ပါဝင်ပါသည်)

ပလက်ဟောင်းပြောင်းလဲမှုများကို ပါဝင်သော စာသားကို ဖြစ်စေပြီး သူတို့၏ တန်ဖိုးများဖြင့် အစားထိုးပေးသည်။
Python `str.format()` ကို အတွင်းပေါင်းအသုံးပြုပါသည်၊ အထက်ပါ [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) ကို အောက်ပါအတိုင်း ကြည့်ပါ။
* သင်သည် `{a:.2f}` ကို အသုံးပြု၍ ပြောင်းလဲမှု ၂ အကြိမ် ပြောင်းလဲပေးနိုင်ပါသည်။
* သင်သည် `{a:05d}` ကို အသုံးပြု၍ အတိုင်းအတွင်း ၅ အကြိမ် အစားထိုးပေးနိုင်ပါသည်။ အောက်ပါ Comfy ဖိုင်နာမည် အဆုံး `ComfyUI_00001_.png` နှင့် ပြောင်းလဲရန် အတွက် အသုံးပြုပါသည်။
* သင်သည် `{ }` ကို သင့်စာသားများတွင် ရေးသားခြင်းဖြင့် (ဥပမာ JSON များအတွက်) အသုံးပြုရန် အတွက် မျှော်လင့်ပေးပါသည်။ မျှော်လင့်ပေးသော `{{ }}` ဖြစ်ပါသည်။

*Search & replace (S&R) syntax* ကို အသုံးပြုပါသည် ဥပမာ `%date:yyyy-MM-dd hh:mm:ss%` နှင့် `%KSampler.seed%` များကို အသုံးပြုပါသည်။
ထို့ကြောင့် သင်သည် အဲဒီနေရာတွင် `GET-node` အဖြစ် အသုံးပြုနိုင်ပါသည်။
"Search & replace" သည် Javascript အကြောင်းအရာတွင် ဖြစ်ပြီး နောက်ပိုင်းတွင် ပြုလုပ်သည်ကို သတိပြုပါ။
