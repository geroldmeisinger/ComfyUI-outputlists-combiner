## ফরমেটেড স্ট্রিং

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow সহ)

প্লেসহোল্ডার ভেরিয়েবল সহ একটি স্ট্রিং তৈরি করে এবং তাদের প্রতিটি মান দিয়ে প্রতিস্থাপন করে।
পাইথন `str.format()` ব্যবহার করে, [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) দেখুন।
* আপনি `{a:.2f}` ব্যবহার করে ফ্লোট মানকে 2 ডিসিমেল পর্যন্ত রাউন্ড করতে পারেন।
* আপনি `{a:05d}` ব্যবহার করে কমফির ফাইলনেম সাফিক্সের সাথে মেলানোর জন্য ৫ টি প্রাথমিক জিরো প্যাড করতে পারেন `ComfyUI_00001_.png`।
* আপনি যদি আপনার স্ট্রিংয়ের মধ্যে `{ }` লিখতে চান (যেমন JSON এর জন্য) তবে আপনাকে দুইবার লিখতে হবে: `{{ }}`।

এছাড়াও *সার্চ এন্ড রিপ্লেস (S&R) সিনট্যাক্স* ব্যবহার করে যেমন `%date:yyyy-MM-dd hh:mm:ss%` এবং `%KSampler.seed%`।
তাই আপনি এটিকে `GET-node` হিসেবেও ব্যবহার করতে পারেন।
দ্রষ্টব্য যে "সার্চ এন্ড রিপ্লেস" জাভাস্ক্রিপ্ট কন্টেক্সটে চলে এবং নোড সঞ্চালনা আগে চলে।
