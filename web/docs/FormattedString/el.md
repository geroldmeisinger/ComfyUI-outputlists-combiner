## Μορφοποιημένη Συμβολοσειρά

![Formatted String](FormattedString/FormattedString.png)

(Συμπεριλαμβανόμενη η ComfyUI workflow)

Δημιουργεί μια συμβολοσειρά που περιέχει μεταβλητές τοποθεσίας και αντικαθιστά τις με τις αντίστοιχες τιμές τους.
Χρησιμοποιεί εσωτερικά την python `str.format()`, δείτε [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Μπορείτε να χρησιμοποιήσετε `{a:.2f}` για να στρογγυλοποιήσετε έναν δεκαδικό αριθμό σε 2 δεκαδικά ψηφία.
* Μπορείτε να χρησιμοποιήσετε `{a:05d}` για να συμπληρώσετε μέχρι 5 μηδενικά στην αρχή για να ταιριάξει με τον suffix του ονόματος αρχείου του comfys `ComfyUI_00001_.png`.
* Αν θέλετε να γράψετε `{ }` μέσα στις συμβολοσειρές σας (π.χ. για JSONs) πρέπει να τα διπλασιάσετε: `{{ }}`.

Εφαρμόζει επίσης τη σύνταξη *αναζήτησης & αντικατάστασης (S&R)* όπως `%date:yyyy-MM-dd hh:mm:ss%` και `%KSampler.seed%`.
Έτσι μπορείτε επίσης να το χρησιμοποιήσετε ως `GET-node`.
Σημειώστε ότι η "αναζήτηση & αντικατάσταση" γίνεται σε πλαίσιο Javascript και εκτελείται πριν την εκτέλεση του κόμβου.
