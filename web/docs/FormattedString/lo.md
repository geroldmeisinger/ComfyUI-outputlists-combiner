## ສະຕິງທີ່ມີຮູບແບບ

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow ໄດ້ຮັບມາດ້ວຍ)

ສ້າງສະຕິງທີ່ມີຕົວແປທີ່ຖືກປ່ຽນແປງ ແລະ ປ່ຽນແປງມັນໃຫ້ເປັນຄ່າທີ່ສັມກັນ.
ໃຊ້ python `str.format()` ສະເໜີໃນຕົວເອງ, ສະແດງເຫັນ [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* ທ່ານສາມາດໃຊ້ `{a:.2f}` ເພື່ອປັບຕົວເລກເປັນສະຕິງເປັນ 2 ຕົວເລກທີ່ມີຕົວແຍກພັນສູນສຳເລັດ.
* ທ່ານສາມາດໃຊ້ `{a:05d}` ເພື່ອປ່ຽນໃຫ້ເປັນ 5 ຕົວເລກສູນສຳເລັດເພື່ອສັມກັບ comfys filename suffix `ComfyUI_00001_.png`.
* ທ່ານສາມາດໃຊ້ `{ }` ເຂົ້າໃນສະຕິງຂອງທ່ານ (ເຊັ່ນໃນ JSONs) ທ່ານຕ້ອງໃຊ້ສະຕິງສອງຄັ້ງ: `{{ }}`.

ໃນເວລານີ້ກໍມີການປ່ຽນແປງ *search & replace (S&R)* ເຊັ່ນ `%date:yyyy-MM-dd hh:mm:ss%` ແລະ `%KSampler.seed%`.
ດັ່ງນັ້ນທ່ານສາມາດໃຊ້ມັນເປັນ `GET-node` ໂດຍທີ່ໃຊ້ໃນການດຳເນີນການ.
ເກັບເຖິງວ່າ "search & replace" ຖືກດຳເນີນໃນ Javascript context ແລະ ດຳເນີນກ່ອນການດຳເນີນການຂອງເຄື່ອງມື.
