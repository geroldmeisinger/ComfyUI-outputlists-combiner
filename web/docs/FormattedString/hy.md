## Ձևաչափված տառանիշ

![Ձևաչափված տառանիշ](FormattedString/FormattedString.png)

(ներառյալ ComfyUI աշխատահետևակայություն)

Ստեղծում է տառանիշ, որը պարունակում է տեղապահիչ փոփոխականներ և փոխարինում է նրանց համապատասխան արժեքներով:
Ներքին միջազգային `str.format()` օգտագործում է Python-ում, տես [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Կարող եք օգտագործել `{a:.2f}` լինի թվի կլորացման համար 2 տասնորդական թվանշաններով:
* Կարող եք օգտագործել `{a:05d}` լինի թվի մինչև 5 նախածանցական զրոներ լրացման համար, որպեսզի համապատասխանի Comfy-ի անունի վերջածանցիներին `ComfyUI_00001_.png`:
* Եթե ցանկանում եք գրել `{ }` ձեր տառանիշերում (օրինակ JSON-ների համար), ապա պետք է կրկնակի գրեք: `{{ }}`.

Նաև կիրառում է *փնտրել և փոխարինել (S&R) սինտաքսը* ինչպես `%date:yyyy-MM-dd hh:mm:ss%` և `%KSampler.seed%`:
Այդպես կարող եք նաև օգտագործել որպես `GET-node`:
Նշեք, որ "փնտրել և փոխարինել" տեղի է ունենում JavaScript կոնտեքստում և աշխատում է ներածման առաջ կատարվող գործողություններից առաջ:
