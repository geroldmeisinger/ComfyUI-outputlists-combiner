## מחרוזת מעוצבת

![Formatted String](FormattedString/FormattedString.png)

(הזרימה ב-ComfyUI נכללת)

יוצר מחרוזת המכילה משתנים מיקומים ומחליף אותם עם הערכים המתאימים להם.
משתמש ב-`str.format()` של Python פנימית, ראה [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* ניתן להשתמש ב-`{a:.2f}` כדי לעגל מספר עשרוני ל-2 ספרות עשרוניות.
* ניתן להשתמש ב-`{a:05d}` כדי להשלים עד 5 אפסים בתחילת המחרוזת כדי להתאים לסופית שם הקובץ של Comfy `ComfyUI_00001_.png`.
* אם ברצונך לכתוב `{ }` בתוך המחרוזות שלך (למשל עבור JSONs) עליך להכפיל אותם: `{{ }}`.

החל גם את תחביר *חיפוש והחלפה (S&R)* כמו `%date:yyyy-MM-dd hh:mm:ss%` ו-`%KSampler.seed%`.
לכן ניתן להשתמש בו גם כ-`GET-node`.
שים לב ש"חיפוש והחלפה" מתבצע בהקשר של Javascript ורץ לפני הרצת צומת.
