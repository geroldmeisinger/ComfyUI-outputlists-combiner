## Форматлаулы текст

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow included)

Урын җитергә һәм уларга соңгы кыйммәтләрне урынлаштырган текст яарат. 
Python `str.format()` кулланып эшләнә,  кара: [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Сез `{a:.2f}` кулланып float'ны 2_decimal урында якынлаштырырга мөмкин.
* Сез `{a:05d}` кулланып comfys файл исеменә 5 нуль булган саен урынлаштырырга мөмкин `ComfyUI_00001_.png`.
* Сез текстта `{ }` язырга теләсәгез (мәгънәлә JSON), аларны икеләп языгыз: `{{ }}`.

Шулай ук *эзләп алыштыру (S&R)* синтаксисын куллана: `%date:yyyy-MM-dd hh:mm:ss%` һәм `%KSampler.seed%`.
Шуңа күрә аны `GET-node` булараҡ да кулланырга мөмкин.
Эзләп алыштыру җитәкле JavaScript контекстында эшләнә һәм нод эшләнә алып барырга кадәр эшләнә.
