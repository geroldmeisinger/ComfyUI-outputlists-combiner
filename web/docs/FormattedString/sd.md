## فارمیٹڈ سٹرنگ

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow شامل آهي)

جيه کي ٺيک ڪري ٿو جيڪو پلیس ھولڊر چارن ہجڻ ۽ اهي سان ملا ٿيل قيمتن سان ڀلي ٿي ٿو. 
پائٿن `str.format()` ننڍي چارن استعمال ڪري ٿو، [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) ڏسو.
* توهان `{a:.2f}` استعمال ڪري سگهيو ٿا چارن ۾ 2 ڊيجٽ گھٽا ٿي سگهيو.
* توهان `{a:05d}` استعمال ڪري سگهيو ٿا چارن ۾ 5 گھٽ ٿيل صفر ڀلي ٿي سگهيو ته comfys filename suffix `ComfyUI_00001_.png` سان ملا ٿي سگهيو.
* جيڪڏهن توهان `{ }` چارن ۾ چاھيو ٿا (جنهن چارن JSON ۾) ته توهان ان کي ٻيهر چارن ۾ چاھيو ٿا: `{{ }}`.

*ساحل ۽ بدلائڻ (S&R) سينٽيکس* جي چارن ۾ چاھيو ٿا جيئن `%date:yyyy-MM-dd hh:mm:ss%` ۽ `%KSampler.seed%`.
ان کي `GET-node` چارن ۾ استعمال ڪري سگهيو ٿا.
ياد رکو "ساحل ۽ بدلائڻ" جا Javascript ساحل ۾ چارن ۽ نوڊ جي اجرا کان اڳ چارن ٿي سگهيو.
