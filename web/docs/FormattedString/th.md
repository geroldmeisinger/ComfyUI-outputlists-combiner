## ตัวแปรสตริงที่จัดรูปแบบ

![Formatted String](FormattedString/FormattedString.png)

(รวม Workflow ของ ComfyUI ไว้ด้วย)

สร้างสตริงที่มีตัวแปรสำรองและแทนที่ด้วยค่าที่สอดคล้องกัน
ใช้ `str.format()` ของ Python ภายใน ดูได้ที่ [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* คุณสามารถใช้ `{a:.2f}` เพื่อปัดเศษทศนิยมของ float ให้เหลือ 2 ตำแหน่ง
* คุณสามารถใช้ `{a:05d}` เพื่อเติมศูนย์นำหน้าจนครบ 5 ตำแหน่ง เพื่อให้สอดคล้องกับส่วนต่อท้ายของชื่อไฟล์ของ ComfyUI `ComfyUI_00001_.png`
* หากคุณต้องการใช้ `{ }` ภายในสตริงของคุณ (เช่น สำหรับ JSON) คุณต้องใช้สองครั้ง: `{{ }}`

ยังใช้ *ไวยากรณ์การค้นหาและแทนที่ (S&R)* เช่น `%date:yyyy-MM-dd hh:mm:ss%` และ `%KSampler.seed%`
ดังนั้น คุณสามารถใช้มันเป็น `GET-node` ได้ด้วย
โปรดทราบว่า "การค้นหาและแทนที่" จะดำเนินการในบริบทของ Javascript และทำงานก่อนการดำเนินการของ node
