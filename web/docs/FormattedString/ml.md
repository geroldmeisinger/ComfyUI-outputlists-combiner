## ഫോർമാറ്റുചെയ്ത സ്ട്രിംഗ്

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI workflow ഉൾപ്പെടുത്തിയിരിക്കുന്നു)

പ്ലേസ്ഹോൾഡർ വേരിയബിൾകൾ ഉള്ള ഒരു സ്ട്രിംഗ് ഉണ്ടാക്കുകയും അവയെ അവയുടെ പ്രത്യേക മൂല്യങ്ങളായി മാറ്റുകയും ചെയ്യുന്നു.
പൈത്തണിലെ `str.format()` ഉപയോഗിച്ച് ഇത് ഇന്റർനല്ലിൽ പ്രവർത്തിക്കുന്നു, [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) കാണുക.
* ഫ്ളോട്ടിനെ 2 ദശാംശങ്ങൾ വരെ മുറിച്ചെടുക്കാൻ `{a:.2f}` ഉപയോഗിക്കാം.
* comfys ഫയൽ പേരിന്റെ അവസാനം `ComfyUI_00001_.png` എന്നതിനു പൊരുത്തപ്പെടുത്താൻ `{a:05d}` ഉപയോഗിക്കാം.
* നിങ്ങൾക്ക് സ്ട്രിംഗുകളിൽ `{ }` എഴുതാൻ ആഗ്രഹിക്കുന്നു (JSON എന്നിവയ്ക്ക് ഉദാഹരണം) അപ്പോൾ അവ ഡബിൾ ചെയ്യേണ്ടതുണ്ട്: `{{ }}`.

അതിനാൽ *തെരയുക അതിലേക്ക് മാറ്റം (S&R) സിന്റാക്സ്* ഉപയോഗിക്കുന്നു ഉദാഹരണത്തിന് `%date:yyyy-MM-dd hh:mm:ss%` എന്നതും `%KSampler.seed%` എന്നതും.
അതിനാൽ ഇത് `GET-node` ആയി ഉപയോഗിക്കാം.
"തെരയുക അതിലേക്ക് മാറ്റം" ജാവാസ്ക്രിപ്റ്റ് കണ്ടെറ്റിൽ നടക്കുന്നു എന്നതാണ് അതിനാൽ നോഡ് പ്രവർത്തനത്തിന് മുൻപാണ് പ്രവർത്തിക്കുന്നത്.
