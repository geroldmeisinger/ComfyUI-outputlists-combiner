## ფორმატირებული სტრიქონი

![Formatted String](FormattedString/FormattedString.png)

(ComfyUI ნამუშევრის მითითებულია)

შექმნის სტრიქონს, რომელიც შეიცავს პლაისელდერის ცვლადებს და ჩანაცვლებს მათ შესაბამის მნიშვნელობებით.
შიდა მნიშვნელობებში გამოიყენება python `str.format()`, იხ. [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* შეიძლება გამოიყენოთ `{a:.2f}` ათწილადის დამრგვალებისთვის 2 ათწილადით.
* შეიძლება გამოიყენოთ `{a:05d}` 5 წინა ნულის დამატებისთვის რომ შეესაბამებოდეს comfys ფაილის სახელის სუფიქსის `ComfyUI_00001_.png`.
* თუ გსურთ დაწეროთ `{ }` თქვენი სტრიქონებში (მაგალითად JSON-ებისთვის), უნდა დააკოპიროთ ისინი: `{{ }}`.

ასევე გამოიყენება *ძებნის და ჩანაცვლების (S&R) სინტაქსი* როგორიცაა `%date:yyyy-MM-dd hh:mm:ss%` და `%KSampler.seed%`.
ამიტომ შეიძლება გამოიყენოთ ასევე `GET-node`-ის სახით.
განიხილეთ, რომ "ძებნა და ჩანაცვლება" ხდება Javascript-ის კონტექსტში და განხორციელდება ნოდის გაშვებამდე.
