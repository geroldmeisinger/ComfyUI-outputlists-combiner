## رشته قالب‌بندی شده

![Formatted String](FormattedString/FormattedString.png)

(فایل کارگاه ComfyUI همراه داده شده است)

این نود یک رشته ایجاد می‌کند که حاوی متغیرهای جایگزین (placeholder) است و آن‌ها را با مقادیر مربوطه جایگزین می‌کند.
از تابع `str.format()` در پایتون به صورت داخلی استفاده می‌کند، برای اطلاعات بیشتر به [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) مراجعه کنید.
* می‌توانید از `{a:.2f}` برای گرد کردن اعداد اعشاری به دو رقم اعشار استفاده کنید.
* می‌توانید از `{a:05d}` برای پر کردن تا 5 صفر در ابتدای عدد برای تطبیق با پسوند نام فایل ComfyUI `ComfyUI_00001_.png` استفاده کنید.
* اگر می‌خواهید در رشته‌های خود `{ }` را بنویسید (مثلاً برای JSON)، باید آن‌ها را دوبار بنویسید: `{{ }}`.

همچنین از سینتکس *جستجو و جایگزینی (S&R)* نیز استفاده می‌کند، مانند `%date:yyyy-MM-dd hh:mm:ss%` و `%KSampler.seed%`.
بنابراین می‌توانید از آن به عنوان یک نود `GET` نیز استفاده کنید.
توجه داشته باشید که "جستجو و جایگزینی" در زمینه جاوااسکریپت انجام می‌شود و قبل از اجرای نود انجام می‌گیرد.
