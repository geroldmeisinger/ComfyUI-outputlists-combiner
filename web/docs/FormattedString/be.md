## Фарматаваны радок

![Formatted String](FormattedString/FormattedString.png)

(Уключаны ComfyUI workflow)

Стварае радок, які ўтрымлівае пераменныя-запаўняльнікі і замяняе іх адпаведнымі значэннямі.
Выкарыстоўвае ў сябе python `str.format()`, глядзі [Python - Format String Syntax](https://docs.python.org/3/library/string.html#format-string-syntax) .
* Вы можаце выкарыстоўваць `{a:.2f}` для акруглення дробнага ліку да 2 дзесятковых.
* Вы можаце выкарыстоўваць `{a:05d}` для дапаўнення да 5 вядучых нуляў для адпаведнасці са суфіксам імёнаў файлаў comfys `ComfyUI_00001_.png`.
* Калі вы хочаце запісаць `{ }` у сваіх радках (напрыклад, для JSON), вам трэба падвоіць іх: `{{ }}`.

Таксама прымяняе сінтаксіс *пошуку і замены (S&R)*, напрыклад `%date:yyyy-MM-dd hh:mm:ss%` і `%KSampler.seed%`.
Такім чынам, вы таксама можаце выкарыстоўваць яго як `GET-node`.
Звярніце ўвагу, што "пошук і замена" адбываецца ў кантэкście Javascript і выконваецца перад выкананнем вузла.
