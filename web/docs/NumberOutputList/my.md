## နံပါတ် OutputList

![Number OutputList](NumberOutputList/NumberOutputList.png)

(ComfyUI workflow ပါဝင်ပါသည်)

နံပါတ်များရှိသော OutputList ကိုဖန်တီးပါမည်။
အမှုတွေ့ရှိသော တန်ဖိုးများနှင့် ပတ်သက်၍ အလုပ်လုပ်ပေးသော [numpy.linspace](https://numpy.org/doc/stable/reference/generated/numpy.linspace.html) ကို အတွေ့အကြုံအရ အသုံးပြုပါမည်။
သင်အတွက် အမှီအခိုက်တွင် နံပါတ်စာရင်းများကို ဖော်ပြရန် အလိုပြောပြောရန် စာရင်းများကို ဖော်ပြပေးသော JSON OutputList ကို စီမံပါ။ ဥပမာ `[1, 42, 123]` အတွက် အသုံးပြုပါ။
`int`, `float`, `string` နှင့် `index` များသည် `is_output_list=True` ကို အသုံးပြုပြီး (symbol `𝌠` ဖြင့် ပြသပါမည်) အတိုင်းအတာတွင် ပေါင်းစပ်ပြီး အကြိမ်ကြိမ် ပြုလုပ်ပါမည်။

### ထောင့်များ

| အမည် | အမျိုးအစား | ဖော်ပြချက် |
| --- | --- | --- |
| `start` | `FLOAT` | အစပိုင်းတန်ဖိုးကို အတိုင်းအတာများကို ဖန်တီးရန်။ |
| `stop` | `FLOAT` | အဆုံးတန်ဖိုး။ `endpoint=include` ဖြစ်ပါက ဤနံပါတ်သည် စာရင်းတွင် ပါဝင်ပါမည်။ |
| `num` | `INT` | စာရင်းတွင်ရှိသော အရာအတွင်းရှိ အရာများ၏ အရေအတွက် (တိုက်ခိုက်မှုအတွက် `step` နှင့် မတူပါ။) |
| `endpoint` | `BOOLEAN` | `stop` တန်ဖိုးသည် အရာများတွင် ပါဝင်မှု သို့မဟုတ် ပိုင်ဆိုင်မှုကို ဆုံးဖြတ်ပါမည်။ |

### ထုတ်လုပ်မှုများ

| အမည် | အမျိုးအစား | ဖော်ပြချက် |
| --- | --- | --- |
| `int` | `INT 𝌠` | တန်ဖိုးကို int သို့ ပြောင်းပေးပါမည် (အောက်ပိုင်းအတိုင်း/floored)။ |
| `float` | `FLOAT 𝌠` | တန်ဖိုးကို float အဖြစ် ပြသပါမည်။ |
| `string` | `STRING 𝌠` | တန်ဖိုးကို float အဖြစ် ပြောင်းပြီး စာသားအဖြစ် ပြသပါမည်။ |
| `index` | `INT 𝌠` | 0..count အထိ အကြိမ်ကြိမ် အသုံးပြုနိုင်သော အညွှန်း။ |
| `count` | `INT` | `num` နှင့် တူပါသည်။ |
