مثال: `[1, 2, 3] x ["A", "B"] = [(1, "A"), (1, "B"), (2, "A"), (2, "B"), (3, "A"), (3, "B")]`

`unzip_a` تا `unzip_d` از `is_output_list=True` (با نماد `𝌠`) استفاده می‌کنند و به ترتیب توسط نودهای مربوطه پردازش می‌شوند.

همه لیست‌ها اختیاری هستند و لیست‌های خالی نادیده گرفته می‌شوند.

در واقع، حاصلضرب کارتزیان را محاسبه می‌کند و هر ترکیب را به عنوان عناصر جداگانه (با `unzip`) خروجی می‌کند؛ در حالی که لیست‌های خالی با `None` جایگزین می‌شوند و در خروجی مربوطه `None` انتقال می‌یابند.

مثال: `[1, 2] x [] x ["A", "B"] x [] = [(1, None, "A", None), (1, None, "B", None), (2, None, "A", None), (2, None, "B", None)]`
