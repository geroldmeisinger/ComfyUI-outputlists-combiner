## Συνδυασμοί των Λίστων Εξόδου

![Συνδυασμοί των Λίστων Εξόδου](CombineOutputLists/CombineOutputLists.png)

(Εργαλείο διαδικτύου ComfyUI περιλαμβάνεται)

Παίρνει μέχρι 4 λίστες εξόδου και δημιουργεί κάθε δυνατό συνδυασμό από αυτές.

Παράδειγμα: `[1, 2, 3] x ["A", "B"] = [(1, "A"), (1, "B"), (2, "A"), (2, "B"), (3, "A"), (3, "B")]`

`unzip_a` .. `unzip_d` χρησιμοποιούν `is_output_list=True` (επισημαίνεται από το σύμβολο `𝌠`) και θα επεξεργαστούν ανακριθείς από τα αντίστοιχα κόμβους.

Όλες οι λίστες είναι επιλογής και άδειες λίστες θα αγνοηθούν.

Επιστημικά υπολογίζει το *καρτεσιανό γινόμενο* και εξάγει κάθε συνδυασμό χωρισμένο σε στοιχεία (`unzip`), ενώ άδειες λίστες θα αντικατασταθούν με μονάδες `None` και θα εκπέμπουν `None` στην αντίστοιχη έξοδο.

Παράδειγμα: `[1, 2] x [] x ["A", "B"] x [] = [(1, None, "A", None), (1, None, "B", None), (2, None, "A", None), (2, None, "B", None)]`
