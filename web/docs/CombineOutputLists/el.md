## Συνδυασμοί OutputLists

![Συνδυασμοί OutputLists](CombineOutputLists/CombineOutputLists.png)

(Συμπεριλαμβάνεται ComfyUI workflow)

Λαμβάνει έως και 4 OutputLists και δημιουργεί όλους τους συνδυασμούς τους.

Παράδειγμα: `[1, 2, 3] x ["A", "B"] = [(1, "A"), (1, "B"), (2, "A"), (2, "B"), (3, "A"), (3, "B")]`

Τα `unzip_a` .. `unzip_d` χρησιμοποιούν `is_output_list=True` (ενδείξεις με το σύμβολο `𝌠`) και θα επεξεργαστούν σειριακά από τα αντίστοιχα nodes.

Όλες οι λίστες είναι προαιρετικές και οι κενές λίστες θα αγνοηθούν.

Τεχνικά υπολογίζει το *καρτεσιανό γινόμενο* και εξάγει κάθε συνδυασμό χωρισμένο στα στοιχεία του (`unzip`), ενώ οι κενές λίστες θα αντικατασταθούν με μονάδες `None` και θα εκπέμπουν `None` στην αντίστοιχη έξοδο.

Παράδειγμα: `[1, 2] x [] x ["A", "B"] x [] = [(1, None, "A", None), (1, None, "B", None), (2, None, "A", None), (2, None, "B", None)]`

### Είσοδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `list_a` | `*` | (προαιρετικό) |
| `list_b` | `*` | (προαιρετικό) |
| `list_c` | `*` | (προαιρετικό) |
| `list_d` | `*` | (προαιρετικό) |

### Έξοδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `unzip_a` | `* 𝌠` | Τιμή των συνδυασμών που αντιστοιχούν στο `list_a`. |
| `unzip_b` | `* 𝌠` | Τιμή των συνδυασμών που αντιστοιχούν στο `list_b`. |
| `unzip_c` | `* 𝌠` | Τιμή των συνδυασμών που αντιστοιχούν στο `list_c`. |
| `unzip_d` | `* 𝌠` | Τιμή των συνδυασμών που αντιστοιχούν στο `list_d`. |
| `index` | `INT 𝌠` | Εύρος 0..count που μπορεί να χρησιμοποιηθεί ως δείκτης. |
| `count` | `INT` | Συνολικός αριθμός συνδυασμών. |
