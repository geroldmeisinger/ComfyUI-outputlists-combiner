## ការ​បញ្ចូល​បញ្ជាក់​លទ្ធផល

![OutputLists Combinations](CombineOutputLists/CombineOutputLists.png)

(រូបរាង ComfyUI បាន​រួម​បញ្ចូល)

យក​ច្បាស់​អតិបរមា 4 បញ្ជាក់​លទ្ធផល ហើយ​បង្កើត​ការ​បញ្ចូល​ទាំងអស់​របស់​វា។

ឧទាហរណ៍: `[1, 2, 3] x ["A", "B"] = [(1, "A"), (1, "B"), (2, "A"), (2, "B"), (3, "A"), (3, "B")]`

`unzip_a` .. `unzip_d` ប្រើ `is_output_list=True` (បង្ហាញ​ដោយ​សញ្ញា `𝌠`) ហើយ​នឹង​ត្រូវ​បាន​ដំណើរការ​តាម​លំដាប់​ដោយ​ផ្ទាល់​ជា​តម្លៃ​នៃ​កម្មវិធី​ត្រូវ​បាន​ដំណើរការ​តាម​ការ​ប្រើ​គ្រឿង​ផ្លូវ​ដែល​ត្រូវ​បាន​បញ្ជាក់​ជា​តម្លៃ​របស់​ការ​បញ្ចូល។

គ្រប់​បញ្ជាក់​លទ្ធផល​គឺ​ជា​ជម្រើស ហើយ​បញ្ជាក់​លទ្ធផល​ទទេះ​នឹង​ត្រូវ​បាន​មិន​ប្រើ​។

នៅ​ចំនុច​បច្ចេកវិទ្យា វាបាន​គណនា​ *ដុំ​ផ្ទៃ​ក្រោម​ការ​ប្រើ​* ហើយ​បញ្ជាក់​លទ្ធផល​បន្ទាប់​ពី​បាន​បំបែក​ទៅ​ជា​ធាតុ​នៃ​វា (`unzip`) ដោយ​សេចក្តី​មិន​មាន​បញ្ជាក់​លទ្ធផល​ទទេះ​នឹង​ត្រូវ​បាន​ជំនួស​ដោយ​ជា​តម្លៃ​របស់ `None` ហើយ​នឹង​បញ្ជាក់​លទ្ធផល​បាន​ផ្ញើ​ជា `None` នៅ​លើ​ចំណុច​ចេញ​ផ្ទាល់​។

ឧទាហរណ៍: `[1, 2] x [] x ["A", "B"] x [] = [(1, None, "A", None), (1, None, "B", None), (2, None, "A", None), (2, None, "B", None)]`
