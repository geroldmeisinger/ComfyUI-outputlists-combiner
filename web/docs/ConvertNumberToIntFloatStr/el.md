Μετατρέπει οποιαδήποτε αριθμητική έκφραση σε `INT`, `FLOAT`, `STRING`.
Χρησιμοποιεί εσωτερικά την `nums_from_string.get_nums` η οποία είναι πολύ αποδεκτή σε αριθμούς. Αποδέχεται ακέραιους αριθμούς, αριθμούς με κλασματική μορφή, ακέραιους ή αριθμούς ως σειρές, σειρές που περιέχουν πολλούς αριθμούς με χαρακτήρες χιλιού.
Χρησιμοποιήστε μια σειρά όπως `123;234;345` για την ταχεία δημιουργία λίστας αριθμών. Δεν χρησιμοποιείτε κομμάτια ως διαχωριστές, αφού μπορεί να ερμηνευτούν ως χιλιού.
