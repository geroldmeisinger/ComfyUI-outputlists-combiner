## Μετατροπή σε Ακέραιο Δεκαδικό Συμβολοσειρά

![Convert To Int Float Str](ConvertNumberToIntFloatStr/ConvertNumberToIntFloatStr.png)

(Συμπεριλαμβανόμενη η ComfyUI workflow)

Μετατρέπει οτιδήποτε αριθμητικό σε `INT` `FLOAT` `STRING`.
Χρησιμοποιεί εσωτερικά την `nums_from_string.get_nums` η οποία είναι πολύ προσιτή στους αριθμούς που δέχεται. Τιδήποτε από πραγματικούς ακέραιους, πραγματικούς δεκαδικούς, ακέραιους ή δεκαδικούς ως συμβολοσειρές, συμβολοσειρές που περιέχουν πολλαπλούς αριθμούς με χωριστικά χιλιάδων.
Χρησιμοποιήστε την συμβολοσειρά `123;234;345` για να δημιουργήσετε γρήγορα μια λίστα αριθμών. Μην χρησιμοποιείτε κόμματα ως διαχωριστικά, καθώς μπορεί να ερμηνευτούν ως χωριστικά χιλιάδων.
Το `int`, `float` και `string` χρησιμοποιούν `is_output_list=True` (δείχνεται με το σύμβολο `𝌠`) και θα επεξεργαστούν σε ακολουθία από τις αντίστοιχες κόμβους.

### Εισόδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `any` | `*` | Οτιδήποτε που μπορεί να μετατραπεί νοητικά σε συμβολοσειρά με αριθμούς που μπορούν να αναλυθούν |

### Έξοδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `int` | `INT 𝌠` | Όλοι οι αριθμοί που βρέθηκαν στη συμβολοσειρά με τα δεκαδικά να έχουν περικοπεί. |
| `float` | `FLOAT 𝌠` | Όλοι οι αριθμοί που βρέθηκαν στη συμβολοσειρά ως δεκαδικοί. |
| `string` | `STRING 𝌠` | Όλοι οι αριθμοί που βρέθηκαν στη συμβολοσειρά ως δεκαδικοί μετατραπέντες σε συμβολοσειρά. |
| `count` | `INT` | Πλήθος των αριθμών που βρέθηκαν στην τιμή. |
