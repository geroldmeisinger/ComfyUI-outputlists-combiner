## បម្លែងទៅជា INT FLOAT STR

![Convert To Int Float Str](ConvertNumberToIntFloatStr/ConvertNumberToIntFloatStr.png)

(បានរួមបញ្ចូល ComfyUI workflow)

បម្លែងអ្វីៗដែលមានលក្ខណៈជាលេខទៅជា `INT` `FLOAT` `STRING`.
ប្រើ `nums_from_string.get_nums` ខាងក្នុង ដែលមានការទទួលយកលេខយ៉ាងច្រើន ដោយមិនមានការកំណត់ច្បាស់លាស់នោះទេ។ អ្វីៗដែលមានលេខច្បាស់លាស់ ដូចជាលេខគត់ លេខទស្សនាិ លេខគត់ ឬលេខទស្សនានៅក្នុងខ្លឹមសារ ឬខ្លឹមសារដែលមានលេខច្រើនជាមួយនឹងការបំបែកលេខរៀង។
ប្រើខ្លឹមសារ `123;234;345` ដើម្បីបង្កើតបញ្ជាក់លេខយ៉ាងឆាប់រហូត។ កុះពុះប្រើចំរុះជាក្បែរ ដោយសារតែវាមានការគិតថាជាការបំបែកលេខរៀង។
`int`, `float` និង `string` ប្រើ `is_output_list=True` (បង្ហាញដោយសញ្ញា `𝌠`) និងនឹងត្រូវបានដំណើរការតាមលំដាប់ដោយតម្លៃដែលត្រូវការដោយស្វ័យប្រវត្តិ។
