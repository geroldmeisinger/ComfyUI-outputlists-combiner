## 정수, 실수, 문자열로 변환

![정수, 실수, 문자열로 변환](ConvertNumberToIntFloatStr/ConvertNumberToIntFloatStr.png)

(ComfyUI workflow 포함)

숫자 같은 모든 것을 `INT` `FLOAT` `STRING` 으로 변환합니다.
내부적으로 `nums_from_string.get_nums` 를 사용하며, 이는 허용하는 숫자에 대해 매우 관대합니다. 실제 정수, 실제 실수, 문자열로된 정수나 실수, 천 단위 구분 기호가 포함된 문자열 등 모든 것을 처리할 수 있습니다.
문자열 `123;234;345` 를 사용하여 숫자 목록을 빠르게 생성할 수 있습니다. 쉼표를 구분 기호로 사용하지 마세요. 천 단위 구분 기호로 해석될 수 있기 때문입니다.
`int`, `float` 및 `string` 은 `is_output_list=True` (기호 `𝌠` 으로 표시됨) 를 사용하며, 해당 노드에 의해 순차적으로 처리됩니다.

### 입력

| 이름 | 유형 | 설명 |
| --- | --- | --- |
| `any` | `*` | 문자열로 변환할 수 있고, 구문 분석 가능한 숫자가 포함된 모든 것 |

### 출력

| 이름 | 유형 | 설명 |
| --- | --- | --- |
| `int` | `INT 𝌠` | 문자열에서 찾은 모든 숫자를 소수점 이하를 제거한 정수로 표시합니다. |
| `float` | `FLOAT 𝌠` | 문자열에서 찾은 모든 숫자를 실수로 표시합니다. |
| `string` | `STRING 𝌠` | 문자열에서 찾은 모든 숫자를 실수로 변환한 후 문자열로 표시합니다. |
| `count` | `INT` | 값에서 찾은 숫자의 개수입니다. |
