## JSON OutputList

![JSON OutputList](JSONOutputList/JSONOutputList.png)

(ComfyUI workflow شامل آهي)

JSON اوبجيكت مانن جي ارري يا ڊيکشنري چي ٻاڱي ڪرڻ جي ذريعي OutputList جو تيار ڪري ٿو.
JSONPath سينٽيکس جي استعمال ڪري ڀر ۾ ڀريل قيمتن کي ٻاڱي ٿو، [JSONPath وٽ وٽيٽيپيڊيا](https://en.wikipedia.org/wiki/JSONPath) ڏسو.
سڀ ملدي قيمتن کي ھڪ ڊگھي فھر ۾ چوٽي ٿي ٿي.
توهان ايندروني ساڳي سترن مان اوبجيكت ٺاهي سگهيو ٿا جيئن `[1, 2, 3]`.
`key`, `value`, `int` ۽ `float` `is_output_list=True` (symbol `𝌠` ڀر ۾ ݰاڻا ٿي) استعمال ڪن ٿا ۽ اهي متن ۾ اچي ٿي ٿا جيئن چي چوٽي ٿي ٿي.

### اِن پُٽ

| نالو | قسم | وضاحت |
| --- | --- | --- |
| `jsonpath` | `STRING` | قيمتن کي ٻاڱي ٿي ٿيڻ لاء JSONPath استعمال ٿي ٿو. |
| `json` | `STRING` | JSON ستر جي چي چوٽي ٿي ٿي ٿو جيئن اوبجيكت. |
| `obj` | `*` | (اختياري) ڪو به اوبجيكت جي چي JSON ستر کي ٻاڱي ٿي ٿو |

### آوٽ پُٽ

| نالو | قسم | وضاحت |
| --- | --- | --- |
| `key` | `STRING 𝌠` | ڊيکشنري جي ڪل ۾ يا ارري جي ايندروني ساڳي ستر (string چي چوٽي ٿي ٿي).  تي چي چوٽي ٿي ٿي ٿو جيئن گلوبال ايندروني ساڳي ستر چي چوٽي ٿي ٿي ٿو. |
| `value` | `STRING 𝌠` | قيمت string چي چوٽي ٿي ٿي. |
| `int` | `INT 𝌠` | قيمت int چي چوٽي ٿي ٿي (اگر ہن چي چوٽي ٿي ٿي نه ٿي سگهيو، 0 ۾ ݰاڻا ٿي). |
| `float` | `FLOAT 𝌠` | قيمت float چي چوٽي ٿي ٿي (اگر ہن چي چوٽي ٿي ٿي نه ٿي سگهيو، 0 ۾ ݰاڻا ٿي). |
| `count` | `INT` | فھر ۾ موجود چي چوٽي ٿي ٿي ٿو |
| `debug` | `STRING` | سڀ ملدي اوبجيكت جي چي چوٽي ٿي ٿي ٿو جيئن فارمٽ ٿيل JSON ستر چي چوٽي ٿي ٿي ٿو |
