## JSON OutputList

![JSON OutputList](JSONOutputList/JSONOutputList.png)

(Συμπεριλαμβανόμενη η ComfyUI workflow)

Δημιουργεί ένα OutputList εξάγοντας πίνακες ή λεξικά από αντικείμενα JSON.
Χρησιμοποιεί τη σύνταξη JSONPath για να εξάγει τις τιμές, δείτε [JSONPath στο Wikipedia](https://en.wikipedia.org/wiki/JSONPath) .
Όλες οι ταίριασμα τιμές επιπεδών σε μια μεγάλη λίστα.
Μπορείτε επίσης να χρησιμοποιήσετε αυτόν τον κόμβο για να δημιουργήσετε αντικείμενα από λεκτικές συμβολοσειρές όπως `[1, 2, 3]`.
Τα `key`, `value`, `int` και `float` χρησιμοποιούν `is_output_list=True` (δείχνεται με το σύμβολο `𝌠`) και θα επεξεργαστούν σε ακολουθία από τους αντίστοιχους κόμβους.

### Εισόδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `jsonpath` | `STRING` | Το JSONPath που χρησιμοποιείται για να εξάγει τις τιμές. |
| `json` | `STRING` | Μια συμβολοσειρά JSON που μεταφράζεται σε ένα αντικείμενο. |
| `obj` | `*` | (προαιρετικό) αντικείμενο οποιουδήποτε τύπου που θα αντικαταστήσει τη συμβολοσειρά JSON |

### Έξοδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `key` | `STRING 𝌠` | Το κλειδί για λεξικά ή ο δείκτης για πίνακες (ως συμβολοσειρά). Τεχνικά είναι ένας καθολικός δείκτης της επεπεδωμένης λίστας για όλα εκτός από τα κλειδιά. |
| `value` | `STRING 𝌠` | Η τιμή ως συμβολοσειρά. |
| `int` | `INT 𝌠` | Η τιμή ως ακέραιος (αν δεν μπορεί να αναλύσει τον αριθμό, προεπιλογή είναι 0). |
| `float` | `FLOAT 𝌠` | Η τιμή ως δεκαδικός (αν δεν μπορεί να αναλύσει τον αριθμό, προεπιλογή είναι 0). |
| `count` | `INT` | Συνολικός αριθμός στοιχείων στην επεπεδωμένη λίστα |
| `debug` | `STRING` | Έξοδος εντοπισμού σφαλμάτων όλων των ταιριασμένων αντικειμένων ως μια μορφοποιημένη συμβολοσειρά JSON |
