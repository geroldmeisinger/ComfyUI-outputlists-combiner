## JSON OutputList

![JSON OutputList](JSONOutputList/JSONOutputList.png)

(ComfyUI workflow ပါဝင်ပါသည်)

JSON အိုဘေ့ဂျက်များမှ အာရေးယား သို့မဟုတ် ဒိုးကျော်များကို ထုတ်ယူ၍ OutputList ကို ဖန်တီးပါသည်။
တန်ဖိုးများကို ထုတ်ယူရန် JSONPath စီရင်နှင့် အသုံးပြုပါသည်၊ အောက်ပါကို ကြည့်ပါ [JSONPath on Wikipedia](https://en.wikipedia.org/wiki/JSONPath) .
အကြိမ်နှစ်ဖြစ်သော တန်ဖိုးများကို တစ်ခုသာ အရှည်တွင် ဖြစ်စေပါမည်။
သင်သည် literal strings များမှ အ obects များကို ဖန်တီးရန် ဤ node ကို အသုံးပြုနိုင်ပါသည်။ `[1, 2, 3]` စသည့် အရာများကို အသုံးပြုပါ။
`key`, `value`, `int` နှင့် `float` များသည် `is_output_list=True` ကို အသုံးပြုပါသည် (symbol `𝌠` ဖြင့် ပြသပါသည်) နှင့် အကြိမ်နှစ်ဖြစ်သော nodes များအောက်ရှိ လုပ်ဆောင်ပါမည်။

### ထုတ်ယူမှုများ

| အမည် | အမျိုးအစား | ဖော်ပြချက် |
| --- | --- | --- |
| `jsonpath` | `STRING` | တန်ဖိုးများကို ထုတ်ယူရန် အသုံးပြုသည့် JSONPath ဖြစ်ပါသည်။ |
| `json` | `STRING` | အိုဘေ့ဂျက်အဖြစ် အသိပေးသော JSON string ဖြစ်ပါသည်။ |
| `obj` | `*` | (option) အမျိုးအစားတစ်ခုသော အိုဘေ့ဂျက် အသုံးပြုပါသည်။ ထုတ်ယူသော JSON string ကို အစားထိုးပါမည်။ |

### ထုတ်ယူမှုများ

| အမည် | အမျိုးအစား | ဖော်ပြချက် |
| --- | --- | --- |
| `key` | `STRING 𝌠` | ဒိုးကျော်များအတွက် ကီးယား သို့မဟုတ် အာရေးယားအတွက် အညို (string အဖြစ်ဖြစ်ပါသည်)။ အဓိကအားဖြင့် တစ်ခုသာ ဖြစ်စေသည့် အရှည်တွင် အိုဘေ့ဂျက်အတွက် global index ဖြစ်ပါသည်။ |
| `value` | `STRING 𝌠` | တန်ဖိုးကို string အဖြစ်ဖြစ်ပါသည်။ |
| `int` | `INT 𝌠` | တန်ဖိုးကို int အဖြစ်ဖြစ်ပါသည် (အမှားတွေ့ရပါက 0 အဖြစ် ပြောင်းပါမည်။) |
| `float` | `FLOAT 𝌠` | တန်ဖိုးကို float အဖြစ်ဖြစ်ပါသည် (အမှားတွေ့ရပါက 0 အဖြစ် ပြောင်းပါမည်။) |
| `count` | `INT` | အရှည်တွင် အိုဘေ့ဂျက်အတွင်း အရာအား အတိုင်းအတာတွင် အရှည်မှတ်ပါသည်။ |
| `debug` | `STRING` | အကြိမ်နှစ်ဖြစ်သော အိုဘေ့ဂျက်များအတွက် ဖော်ပြသော JSON string ဖြစ်ပါသည်။ |
