## JSON OutputList

![JSON OutputList](JSONOutputList/JSONOutputList.png)

(បានរួមបញ្ចូល ComfyUI workflow)

បង្កើត OutputList ដោយការវាស់ប្រមាណវិធីបញ្ជាក់ឬវត្ថុនៃ JSON objects ។
ប្រើសំណុំសំណុំ JSONPath ដើម្បីវាស់ប្រមាណវិធីតម្លៃ មើល [JSONPath នៅលើ Wikipedia](https://en.wikipedia.org/wiki/JSONPath) ។
តម្លៃដែលត្រូវបានស្វែងរកទាំងអស់ត្រូវបានធ្វើឱ្យជាបញ្ជាក់ទៅជាបញ្ជាក់មួយ។
អ្នកក៏អាចប្រើ node នេះដើម្បីបង្កើតវត្ថុពីខ្លឹមសារឯកសារដូចជា `[1, 2, 3]` ។
`key`, `value`, `int` និង `float` ប្រើ `is_output_list=True` (បញ្ជាក់ដោយសញ្ញា `𝌠`) ហើយនឹងត្រូវបានដំណើរការតាមលំដាប់ដោយ node ដែលពាក់ព័ន្ធ។

### បញ្ចូល

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
| --- | --- | --- |
| `jsonpath` | `STRING` | JSONPath ដែលប្រើដើម្បីវាស់ប្រមាណតម្លៃ។ |
| `json` | `STRING` | ខ្លឹមសារ JSON ដែលត្រូវបានប្ដូរទៅជាវត្ថុ។ |
| `obj` | `*` | (ជាក់ស្តែង) វត្ថុនៃប្រភេទណាមួយដែលនឹងជំនួសខ្លឹមសារ JSON |

### លទ្ធផល

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
| --- | --- | --- |
| `key` | `STRING 𝌠` | កូនសោសម្រាប់វត្ថុនិម្មិត ឬសេចក្តីសម្រាប់អារេ (ជាខ្លឹមសារ)។ ការប្រើប្រាស់នេះគឺជាកូនសោសំរាប់បញ្ជាក់មួយដែលត្រូវបានធ្វើឱ្យជាបញ្ជាក់ទៅជាបញ្ជាក់មួយសម្រាប់អារេនិម្មិត។ |
| `value` | `STRING 𝌠` | តម្លៃជាខ្លឹមសារ។ |
| `int` | `INT 𝌠` | តម្លៃជាខ្លឹមសារ (ប្រសិនបើមិនអាចវាស់ប្រមាណវិធីចំនួននោះបាន នោះនឹងត្រូវបានកំណត់ទៅជា 0)។ |
| `float` | `FLOAT 𝌠` | តម្លៃជាខ្លឹមសារ (ប្រសិនបើមិនអាចវាស់ប្រមាណវិធីចំនួននោះបាន នោះនឹងត្រូវបានកំណត់ទៅជា 0)។ |
| `count` | `INT` | ចំនួនសរុបនៃធាតុនៅក្នុងបញ្ជាក់មួយ |
| `debug` | `STRING` | លទ្ធផលកំហុសនៃវត្ថុនៃការស្វែងរកទាំងអស់ជាខ្លឹមសារ JSON ដែលបានរចនាប័ទ្ម |
