## Load Any File

![Load Any File](LoadAnyFile/LoadAnyFile.png)

(ComfyUI workflow ഉൾപ്പെടുത്തിയിരിക്കുന്നു)

ഏതെങ്കിലും ടെക്സ്റ്റ് അല്ലെങ്കിൽ ബൈനറി ഫയൽ ലോഡ് ചെയ്യുന്നു, അതിന്റെ ഉള്ളടക്കം സ്ട്രിംഗായോ ബേസ്64 സ്ട്രിംഗായോ നൽകുന്നു. ഇത് `IMAGE` ആയി ലോഡ് ചെയ്യാൻ ശ്രമിക്കുന്നു. മെറ്റാഡേറ്റയും ലോഡ് ചെയ്യാൻ ശ്രമിക്കുന്നു.

`filepath` എന്നത് ComfyUI യുടെ അനോട്ടേറ്റ് ഫയൽപാത്തുകൾ `[input]` `[output]` അല്ലെങ്കിൽ `[temp]` ഉപയോഗിക്കുന്നു.
`filepath` എന്നത് ഗ്ലോബ്-പാറ്റേൺ വിപുലീകരണം പിന്തുടർന്നു `subdir/**/*.png`.
അടിസ്ഥാനത്തിൽ python യുടെ [glob.iglob](https://docs.python.org/3/library/glob.html#glob.iglob) ഉപയോഗിക്കുന്നു.

`metadata` എന്നത് `exiftool` ഇന്‍സ്റ്റാളുചെയ്തിട്ടുണ്ടെങ്കിൽ `PATH` ലെ ലഭ്യമാണെങ്കിൽ അത് കോളുചെയ്യുന്നു, അല്ലെങ്കിൽ `PIL.Image.info` എന്നത് ഫോൾബാക്കായി ഉപയോഗിക്കുന്നു.

സുരക്ഷാ കാരണങ്ങളാൽ മാത്രമേ താഴെപ്പറയുന്ന ഡയറക്ടറികൾ പിന്തുടർന്നുള്ളൂ: `[input] [output] [temp]`.
കാര്യം പ്രവർത്തിക്കാൻ പ്രത്യേകമായി ഫയലുകളുടെ എണ്ണം പരിമിതപ്പെടുത്തിയിരിക്കുന്നു: 1024.

### ഇൻപുട്ടുകൾ

| നാമം | തരം | വിവരണം |
| --- | --- | --- |
| `filepath` | `STRING` | ബേസ് ഡയറക്ടറി അതിന്റെ ഡീഫോൾട്ട് `[input]` ഉപയോക്തൃ ഡയറക്ടറി. ഗ്ലോബ്-പാറ്റേൺ വിപുലീകരണം `subdir/**/*.png` പിന്തുടർന്നു. വേറൊരു ComfyUI ഉപയോക്തൃ ഡയറക്ടറി നിർണ്ണയിക്കാൻ ` [input]` ` [output]` അല്ലെങ്കിൽ ` [temp]` (ഓരോ വിട്ടുകളിലെ വെള്ളം ഉപയോഗിച്ച്!) ഉപയോഗിക്കുക. |

### ഔട്ട്പുട്ടുകൾ

| നാമം | തരം | വിവരണം |
| --- | --- | --- |
| `content` | `STRING` | ടെക്സ്റ്റ് ഫയലുകൾക്കായി ഫയൽ ഉള്ളടക്കം, ബൈനറി ഫയലുകൾക്കായി ബേസ്64. |
| `image` | `IMAGE` | ഇമേജ് ബാച്ച് ടെൻസർ. |
| `mask` | `MASK` | മാസ്ക് ബാച്ച് ടെൻസർ. |
| `metadata` | `STRING` | ExifTool ലെ എക്സിഫ് ഡേറ്റ. `exiftool` കമാൻഡ് `PATH` ൽ ലഭ്യമായിരിക്കണം. |
