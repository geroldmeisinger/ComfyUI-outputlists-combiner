## KSampler உடனடி சேமி

![KSampler உடனடி சேமி](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow கூட்டும்)

இயல்புநிலை `CheckpointLoader`, `KSampler`, `VAE Decode` மற்றும் `Save Image` களை ஒரு கூட்டு செயலாக செயல்படுத்தும் நோட் விரிவாக்கம். இது கட்டிடங்களுக்கு இடைநிலை படங்களை உடனடியாக சேமிக்க விரும்பினால் பயன்படும்.

*"ஒரு படத்தை சேமிக்க கூடிய கஸ்டம் KSampler? இப்போது நான் தவிர்த்து விட விரும்பிய முறையை நான் தான் ஆகிவிட்டேன்!"*

### உள்ளீடுகள்

| பெயர் | வகை | விளக்கம் |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | ஏற்ற தேர்வு மாதிரியின் பெயர். |
| `positive` | `STRING` | படத்தில் காண்பிக்க விரும்பும் பண்புகளை விளக்கும் காரணி. |
| `negative` | `STRING` | படத்தில் காண்பிக்க விரும்பாத பண்புகளை விளக்கும் காரணி. |
| `latent_image` | `LATENT` | தீவிரமாக்க தேவையான முன்படம். |
| `seed` | `INT` | தொடர்புடைய மாற்றத்தை உருவாக்க பயன்படும் சீட். |
| `steps` | `INT` | தீவிரமாக்கல் செயலில் பயன்படும் படிகளின் எண்ணிக்கை. |
| `cfg` | `FLOAT` | தொடர்புடைய வழிமுறை அளவு மாற்றத்தை சமநிலைப்படுத்துகிறது, இது உருவாக்கல் மற்றும் விளக்கம் மாற்றத்தை செயல்படுத்துகிறது. அதிகமான மதிப்புகள் விளக்கத்தை அதிகமாக பொருந்தும் ஆனால் அதிகமான மதிப்புகள் தீவிரமாக்கல் தீமையை செயல்படுத்துகிறது. |
| `sampler_name` | `COMBO` | மாற்றத்தை கண்டுபிடிக்கும் அல்கோரிதம், இது தீவிரமாக்கல் தீமை, வேகம் மற்றும் பாணியை செயல்படுத்துகிறது. |
| `scheduler` | `COMBO` | தீவிரமாக்கல் முறையை செயல்படுத்தும் தீவிரமாக்கல் முறையை கட்டுப்படுத்துகிறது. |
| `denoise` | `FLOAT` | தீவிரமாக்கல் செயல் பொருந்தும் அளவு, குறைந்த மதிப்புகள் முதல் படத்தின் கட்டமைப்பை பாதுகாக்கிறது மற்றும் படத்தில் படம் மாற்றத்தை செயல்படுத்துகிறது. |
| `filename_prefix` | `STRING` | சேமிக்க வேண்டிய கோப்பின் முன்னொட்டு. இது %date :yyyy-MM-dd% அல்லது %Empty Latent Image.width% போன்ற தரவு வடிவமைப்பு தகவல்களை கொண்டிருக்கலாம். |
