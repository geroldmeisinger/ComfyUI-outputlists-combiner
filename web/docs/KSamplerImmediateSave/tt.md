## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow-ы кертелән)

Бергә эшләп үткәрү өчен стандарт `CheckpointLoader`, `KSampler`, `VAE Decode` һәм `Save Image` нодаларын күпәйтү. Был гади күрсәткәчләрне тәкъдим итәргә һәм гридләр өчен аралык сурәтләрне сакларга кирәк булганда кулланылган.

*"Сурәт сакларга китергән күпәйтүләрнең күпәйтүе? Моннан соң манда үземнең тәкъдим итәргә теләгән нәрсәгә әйләнеп куелган!"*

### Кертү

| Исем | Төр | Тасвирлау |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | Ўкытудан һуң күрсәтелгән модельнең исеме. |
| `positive` | `STRING` | Сурәткә өстәргә теләгән атрибутларны тасвирлау. |
| `negative` | `STRING` | Сурәттән алып тәкъдим итәргә теләгән атрибутларны тасвирлау. |
| `latent_image` | `LATENT` | Шыныч сурәт. |
| `seed` | `INT` | Шумы барлыкка китерү өчен кулланылган күрсәткәч. |
| `steps` | `INT` | Шумы алып тәкъдим итү процессында кулланылган күрсәткәч. |
| `cfg` | `FLOAT` | Классификаторыз-гидранция күрсәткәче, яһәт һәм тәкъдим итүнең тәртип бәйләнешен тәкъдим итә. Җитәрле бик күп күрсәткәчләр сурәтнең саплыгына негатив тәсир итә ала. |
| `sampler_name` | `COMBO` | Әлеге алгоритм һәрвакыт кулланыла, бу сурәтнең саплыгы, тизлеге һәм стильне үзгәртә ала. |
| `scheduler` | `COMBO` | Сурәтне төзәтеү өчен шумы алып тәкъдим итү. |
| `denoise` | `FLOAT` | Алып тәкъдим итү күрсәткәче, бик күп күрсәткәчләр сурәтнең структурасын саклый ала һәм сурәттән сурәткә күчешне рөхсәт итә. |
| `filename_prefix` | `STRING` | Саклау өчен файл исеме. Был %date :yyyy-MM-dd% яки %Empty Latent Image.width% кебек үзлекләрне кертә ала һәм нодалардан күрсәткәчләрне алып күтәрә ала. |

### Чыгыш

| Исем | Төр | Тасвирлау |
| --- | --- | --- |
| `image` | `IMAGE` | Сурәтне төзәтү. |
