## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow รวมอยู่ในไฟล์)

การขยาย Node ของ `CheckpointLoader` เดิม, `KSampler`, `VAE Decode` และ `Save Image` เพื่อให้ดำเนินการเป็นหนึ่งชุด
มีประโยชน์หากคุณต้องการบันทึกภาพระหว่างการประมวลผลสำหรับกริดทันที

*"KSampler แบบกำหนดเองเพื่อบันทึกภาพ? ตอนนี้ฉันกลายเป็นสิ่งที่ฉันต้องการกำจัด!"*

### ป้อนข้อมูล

| ชื่อ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | ชื่อของ checkpoint (model) ที่จะโหลด |
| `positive` | `STRING` | การกำหนดเงื่อนไขที่อธิบายคุณสมบัติที่คุณต้องการให้อยู่ในภาพ |
| `negative` | `STRING` | การกำหนดเงื่อนไขที่อธิบายคุณสมบัติที่คุณต้องการให้ไม่อยู่ในภาพ |
| `latent_image` | `LATENT` | ภาพลатเทนต์ที่จะลดเสียงรบกวน |
| `seed` | `INT` | ค่า seed แบบสุ่มที่ใช้ในการสร้างเสียงรบกวน |
| `steps` | `INT` | จำนวนขั้นตอนที่ใช้ในการลดเสียงรบกวน |
| `cfg` | `FLOAT` | สเกล Classifier-Free Guidance ควบคุมความคิดสร้างสรรค์และความถูกต้องตาม prompt ค่าที่สูงจะทำให้ภาพใกล้เคียงกับ prompt มากขึ้น แต่ค่าที่สูงเกินไปจะส่งผลเสียต่อคุณภาพ |
| `sampler_name` | `COMBO` | อัลกอริทึมที่ใช้ในการสุ่ม ซึ่งสามารถส่งผลต่อคุณภาพ ความเร็ว และสไตล์ของผลลัพธ์ที่สร้างขึ้น |
| `scheduler` | `COMBO` | Scheduler ควบคุมวิธีการกำจัดเสียงรบกวนเพื่อสร้างภาพ |
| `denoise` | `FLOAT` | ปริมาณการลดเสียงรบกวนที่ใช้ ค่าที่ต่ำจะรักษาโครงสร้างของภาพต้นฉบับ ทำให้สามารถใช้ในการสุ่มภาพจากภาพอื่นได้ |
| `filename_prefix` | `STRING` | คำนำหน้าของไฟล์ที่จะบันทึก อาจรวมข้อมูลการจัดรูปแบบ เช่น %date:yyyy-MM-dd% หรือ %Empty Latent Image.width% เพื่อใส่ค่าจาก node |

### ผลลัพธ์

| ชื่อ | ประเภท | คำอธิบาย |
| --- | --- | --- |
| `image` | `IMAGE` | ภาพที่ถูกถอดรหัสแล้ว |
