## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow килеше)

Килешеүҙәр `CheckpointLoader`, `KSampler`, `VAE Decode` һәм `Save Image` ноддарын ҡыҙыҡ булдырып, уларҙы бер үҙен һөрөм булдырып ҡуйырға мөмкин.
Был ҡыҙыҡ булдырылған һөрөмҙәрҙән гриды һөрөм булдырып ҡуйырға мөмкин.

*"Күрһәтелгән һөрөм булдырылған һөрөм булдырып ҡуйырға мөмкин? Менән һөрөм булдырылған һөрөм булдырып ҡуйырға мөмкин!"*

### Керештәр

| Исем | Төр | Сипатлама |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | Алып ҡуйылған һөрөм (модель) исеме. |
| `positive` | `STRING` | һөрөм булдырылған һөрөмҙәрҙең асыл һөрөм булдырылған һөрөм. |
| `negative` | `STRING` | һөрөм булдырылған һөрөмҙәрҙең һөрөм булдырылған һөрөм. |
| `latent_image` | `LATENT` | һөрөм булдырылған һөрөм. |
| `seed` | `INT` | һөрөм булдырылған һөрөм. |
| `steps` | `INT` | һөрөм булдырылған һөрөм. |
| `cfg` | `FLOAT` | һөрөм булдырылған һөрөм. |
| `sampler_name` | `COMBO` | һөрөм булдырылған һөрөм. |
| `scheduler` | `COMBO` | һөрөм булдырылған һөрөм. |
| `denoise` | `FLOAT` | һөрөм булдырылған һөрөм. |
| `filename_prefix` | `STRING` | һөрөм булдырылған һөрөм. |

### Шығыштар
