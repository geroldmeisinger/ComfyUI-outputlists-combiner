## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ներառյալ ComfyUI աշխատահետևակայություն)

Հանգույցի հանրահայտացման համար սովորական `CheckpointLoader`, `KSampler`, `VAE Decode` և `Save Image` հանգույցների համար, որոնք կատարվում են մի անգամ:
Սա հարմար է, եթե ցանկանում եք անմիջապես պահպանել միջնական նկարները ցանցերի համար:

*"Ընդամենը մի հանգույց KSampler նկարը պահպանելու համար? Այժմ ես դարձել եմ այն բանը, ինչը ուզում էի կայացնել:"*

### Մուտքեր

| Անուն | Տիպ | Նկարագրություն |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | Ներբեռնվող կետի (մոդելի) անունը: |
| `positive` | `STRING` | Պայմանագրությունը, որն ընդգրկում է նկարում ներկայացվող հատկանիշները: |
| `negative` | `STRING` | Պայմանագրությունը, որն ընդգրկում է նկարում չներկայացվող հատկանիշները: |
| `latent_image` | `LATENT` | Դենոիզացվող լատենտ նկարը: |
| `seed` | `INT` | Շարահյուսական կետը, որն օգտագործվում է անապատակ ձայնի ստեղծման համար: |
| `steps` | `INT` | Դենոիզացման գործընթացում օգտագործվող քայլերի քանակը: |
| `cfg` | `FLOAT` | Կլասսիֆիկատորի առանց ուղղության սահմանաչափը համատեղում է ստեղծագործությունը և հարցումներին հետագային համապատասխանությունը: Ավելի բարձր արժեքները արդյունքում ստեղծում են նկարներ, որոնք ավելի շատ համապատասխանում են հարցումներին, սակայն ավելի բարձր արժեքները կարող են բացասական ազդեցություն ունենալ որոշակի որակի վրա: |
| `sampler_name` | `COMBO` | Նմանական ալգորիթմը, որն օգտագործվում է նմանական գործընթացում, որը կարող է ազդել արդյունքների որակի, արագության և ճարտարապետական արտահայտության վրա: |
| `scheduler` | `COMBO` | Պլանավորիչը կարգավորում է անապատակ ձայնի առաջացման գործընթացը նկարի ձևավորման համար: |
| `denoise` | `FLOAT` | Դենոիզացման մակարդակը, ցածր արժեքները պահպանում են սկզբնական նկարի կառուցվածքը, թույլատրելով նկարից նկար նմանական գործընթացը: |
| `filename_prefix` | `STRING` | Պահպանվող ֆայլի նախածայնը: Այն կարող է պարունակել ձևավորման տեղեր, օրինակ %date :yyyy-MM-dd% կամ %Empty Latent Image.width% տարրերը հանգույցներից արժեքներ ներմուծելու համար: |
