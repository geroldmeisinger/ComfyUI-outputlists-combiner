## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow ပါဝင်ပါသည်)

အများအရေး `CheckpointLoader`, `KSampler`, `VAE Decode` နှင့် `Save Image` များကို တစ်ခုတည်း လုပ်ဆောင်ရန် အတွက် ဆိုဒ်ကို ခွဲခြောက်ပေးပါသည်။ အမှုတွင် ဂရစ်များအတွက် အလုပ်အမှုတွင် ပြောင်းလဲမှုများကို သိမ်းဆည်းရန် အသုံးပြုနိုင်ပါသည်။

*"ပုံကိုသိမ်းဆည်းရန် အတွက် ကိုယ်ပိုင် KSampler တစ်ခုလိုလား။ အခု ကိုယ်တိုင် အမှုတွင် ရှာဖွေခဲ့သော အမှုတွင် ဖြစ်သွားလာပါသည်!"*

### ထုတ်လုပ်မှုများ

| နာမည် | အမျိုးအစား | ဖော်ပြချက် |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | တွင်းကိုယ်တိုင်အမျိုးအစားကို တွင်းကိုယ်တိုင်သိမ်းဆည်းပေးမည့် အမည်။ |
| `positive` | `STRING` | ပုံတွင် ပါဝင်သည့် အမျိုးအစားများကို ဖော်ပြသော အချက်အလက်။ |
| `negative` | `STRING` | ပုံတွင် မပါဝင်သည့် အမျိုးအစားများကို ဖော်ပြသော အချက်အလက်။ |
| `latent_image` | `LATENT` | မှုတွင် အကြောင်းအရာများကို သိမ်းဆည်းပေးသည့် ပုံ။ |
| `seed` | `INT` | အကြောင်းအရာများကို ဖြစ်ပေါ်စေသည့် အကြိမ်အများအရေး။ |
| `steps` | `INT` | အကြောင်းအရာများကို သိမ်းဆည်းရန် အကြိမ်အများအရေး။ |
| `cfg` | `FLOAT` | အမှုတွင် အကြိမ်အများအရေးအတွက် အကြိမ်အများအရေးများကို အမှုတွင် ပေါင်းစပ်သည့် အကြိမ်အများအရေး။ |
| `sampler_name` | `COMBO` | အကြိမ်အများအရေးများကို အသုံးပြုသည့် အကြိမ်အများအရေး။ အကြိမ်အများအရေးများကို ပေါင်းစပ်သည့် အကြိမ်အများအရေး။ |
| `scheduler` | `COMBO` | အကြိမ်အများအရေးများကို အသုံးပြုသည့် အကြိမ်အများအရေး။ အကြိမ်အများအရေးများကို ပေါင်းစပ်သည့် အကြိမ်အများအရေး။ |
| `denoise` | `FLOAT` | အကြိမ်အများအရေးများကို အသုံးပြုသည့် အကြိမ်အများအရေး။ အကြိမ်အများအရေးများကို ပေါင်းစပ်သည့် အကြိမ်အများအရေး။ |
| `filename_prefix` | `STRING` | သိမ်းဆည်းမည့် ဖိုင်အတွက် စတီးသော အမည်။ အကြိမ်အများအရေးများကို ပေါင်းစပ်သည့် အကြိမ်အများအရေး။ |
