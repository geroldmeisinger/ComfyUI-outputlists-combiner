## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow ഉൾപ്പെടുത്തിയിരിക്കുന്നു)

സാധാരണ `CheckpointLoader`, `KSampler`, `VAE Decode` എന്നിവയുടെ നോഡ് വിപുലീകരണം, അതുപോലെ `Save Image` നോഡിനെ ഒരു കൂട്ടം പ്രക്രിയയായി പ്രവർത്തിപ്പിക്കാൻ കഴിയുന്ന രീതിയിൽ. ഈ നോഡ് ഗ്രിഡിനായി ഇടയ്ക്കുള്ള ഇമേജുകൾ ഉടൻ സൂക്ഷ്മമായി സേവ് ചെയ്യാൻ ആഗ്രഹിക്കുന്നവർക്ക് ഉപകാരപ്രദമാണ്.

*"ഒരു ഇമേജ് സേവ് ചെയ്യാൻ ഒരു കസ്റ്റം KSampler? ഇപ്പോൾ ഞാൻ നിലവിലെ നിയന്ത്രണം ചെയ്യാൻ ആഗ്രഹിച്ച സാധനമായി നിലവിലുണ്ടായിരിക്കുന്നു!"*

### ഇൻപുട്ടുകൾ

| നാമം | തരം | വിവരണം |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | ലോഡ് ചെയ്യാൻ ആവശ്യമുള്ള ചെക്ക്പോയിന്റ് (model) നാമം. |
| `positive` | `STRING` | ഇമേജിൽ ഉൾപ്പെടുത്താൻ ആഗ്രഹിക്കുന്ന വിശേഷണങ്ങൾ വിവരിക്കുന്ന കണ്ടെയിനിംഗ്. |
| `negative` | `STRING` | ഇമേജിൽ നിന്നും നീക്കം ചെയ്യാൻ ആഗ്രഹിക്കുന്ന വിശേഷണങ്ങൾ വിവരിക്കുന്ന കണ്ടെയിനിംഗ്. |
| `latent_image` | `LATENT` | ഡിനോയ്സ് ചെയ്യാൻ ഉപയോഗിക്കുന്ന ലാറ്റൻറ് ഇമേജ്. |
| `seed` | `INT` | നോയ്സ് സൃഷ്ടിക്കാൻ ഉപയോഗിക്കുന്ന രണ്ടാം ക്രമം നിയന്ത്രണം. |
| `steps` | `INT` | ഡിനോയ്സിംഗ് പ്രക്രിയയിൽ ഉപയോഗിക്കുന്ന സ്റ്റെപ്പുകൾ. |
| `cfg` | `FLOAT` | ക്ലാസിഫയർ-ഫ്രീ ഗൈഡൻസ് സ്കെയിൽ സൃഷ്ടിയും പ്രോമ്പ്റ്റിനെ പിന്തുടർന്നുള്ള വിശേഷതകളും തുല്യമാക്കുന്നു. കൂടുതൽ മൂല്യങ്ങൾ പ്രോമ്പ്റ്റിനെ കൂടുതൽ വിശേഷിപ്പിക്കുന്ന രീതിയിൽ ഇമേജുകൾ ഉണ്ടാക്കുന്നു, എന്നാൽ അതിന്റെ ഉചിതമായ മൂല്യങ്ങൾ പ്രത്യേകം കൂടുതൽ മൂല്യങ്ങളാക്കിയാൽ ഗുണം കുറയ്ക്കുന്നു. |
| `sampler_name` | `COMBO` | സാമ്പിളിംഗിൽ ഉപയോഗിക്കുന്ന അൽഗോരിതം, ഇത് ഉത്പാദിപ്പിക്കുന്ന വിവരണത്തിന്റെ ഗുണം, വേഗത, ശൈലി മാറ്റാൻ കഴിയുന്നു. |
| `scheduler` | `COMBO` | സ്കെഡ്യൂലർ നോയ്സ് ക്രമേണ നീക്കം ചെയ്യാൻ ഉപയോഗിക്കുന്നു, അത് ഇമേജ് ഉണ്ടാക്കുന്നതിന് ഉപയോഗിക്കുന്നു. |
| `denoise` | `FLOAT` | ഡിനോയ്സിംഗ് പ്രക്രിയയിൽ ഉപയോഗിക്കുന്ന മൂല്യം, കുറഞ്ഞ മൂല്യങ്ങൾ ആദ്യ ഇമേജിന്റെ രൂപം പാടിയെടുത്ത് ഇമേജ് ഇമേജിൽ മാറ്റം ചെയ്യാൻ അനുവദിക്കുന്നു. |
| `filename_prefix` | `STRING` | സേവ് ചെയ്യുന്ന ഫയൽ പ്രിഫിക്സ്. ഇതിൽ %date :yyyy-MM-dd% അല്ലെങ്കിൽ %Empty Latent Image.width% എന്നിവയുപയോഗിച്ച് നോഡുകളിൽ നിന്നുള്ള മൂല്യങ്ങൾ ഉൾപ്പെടുത്താവുന്നു. |
