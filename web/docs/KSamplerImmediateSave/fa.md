## ذخیره فوری KSampler

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(فایل کارگاه ComfyUI همراه داده شده است)

گسترش نود پیش‌فرض `CheckpointLoader`، `KSampler`، `VAE Decode` و `Save Image` برای پردازش به صورت یکی.
این کار زمانی مفید است که می‌خواهید تصاویر میانی برای گریدها را فوراً ذخیره کنید.

*"یک KSampler سفارشی فقط برای ذخیره یک تصویر؟ حالا من شکلی شدم که خودم را می‌خواستم نابود کنم!"*

### ورودی‌ها

| نام | نوع | توضیح |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | نام چک‌پوینت (مدل) برای بارگذاری. |
| `positive` | `STRING` | شرایطی که ویژگی‌هایی را که می‌خواهید در تصویر شامل شوند توصیف می‌کند. |
| `negative` | `STRING` | شرایطی که ویژگی‌هایی را که می‌خواهید از تصویر حذف شوند توصیف می‌کند. |
| `latent_image` | `LATENT` | تصویر لاتنت برای کاهش نویز. |
| `seed` | `INT` | دانه تصادفی استفاده شده برای ایجاد نویز. |
| `steps` | `INT` | تعداد مرحله‌های استفاده شده در فرآیند کاهش نویز. |
| `cfg` | `FLOAT` | مقیاس هدایت‌کننده بدون دسته‌بندی تعادل بین خلاقیت و پیروی از پرامپت را تعیین می‌کند. مقادیر بالاتر منجر به تصاویری می‌شوند که بیشتر با پرامپت مطابقت دارند، اما مقادیر بسیار بالا می‌توانند کیفیت را منفی تأثیر بگذارند. |
| `sampler_name` | `COMBO` | الگوریتم استفاده شده در نمونه‌برداری، این می‌تواند بر کیفیت، سرعت و سبک خروجی تولید شده تأثیر بگذارد. |
| `scheduler` | `COMBO` | برنامه‌ریز چگونگی حذف تدریجی نویز برای تشکیل تصویر را کنترل می‌کند. |
| `denoise` | `FLOAT` | مقدار کاهش نویز اعمال شده، مقادیر پایین‌تر ساختار تصویر اولیه را حفظ می‌کنند و اجازه نمونه‌برداری از تصویر به تصویر را می‌دهند. |
| `filename_prefix` | `STRING` | پیشوند فایل برای ذخیره. این ممکن است شامل اطلاعات قالب‌بندی مانند %date:yyyy-MM-dd% یا %Empty Latent Image.width% باشد تا مقادیری از نودها را شامل شوند. |

### خروجی‌ها

| نام | نوع | توضیح |
| --- | --- | --- |
| `image` | `IMAGE` | تصویر کدگشایی شده. |
