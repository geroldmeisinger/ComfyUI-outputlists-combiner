## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(Συμπεριλαμβανόμενη η ComfyUI workflow)

Επέκταση κόμβου του προεπιλεγμένου `CheckpointLoader`, `KSampler`, `VAE Decode` και `Save Image` για να επεξεργαστούν ως ένα.
Αυτό είναι χρήσιμο αν θέλετε να αποθηκεύσετε τις ενδιάμεσες εικόνες για πλέγματα αμέσως.

*"Ένας προσαρμοσμένος KSampler μόνο για να αποθηκεύσει μια εικόνα; Τώρα έγινα το πράγμα που είχα αναζητήσει να καταστρέψω!"*

### Εισόδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | Το όνομα του σημείου ελέγχου (μοντέλου) που θα φορτωθεί. |
| `positive` | `STRING` | Η συνθήκη που περιγράφει τα χαρακτηριστικά που θέλετε να συμπεριλάβετε στην εικόνα. |
| `negative` | `STRING` | Η συνθήκη που περιγράφει τα χαρακτηριστικά που θέλετε να εξαιρέσετε από την εικόνα. |
| `latent_image` | `LATENT` | Η εικόνα latent που θα αποκαθαριστεί. |
| `seed` | `INT` | Το τυχαίο seed που χρησιμοποιείται για τη δημιουργία του θορύβου. |
| `steps` | `INT` | Ο αριθμός των βημάτων που χρησιμοποιούνται στη διαδικασία αποκαθάρισης. |
| `cfg` | `FLOAT` | Το κλίμακα Classifier-Free Guidance ισορροπεί τη δημιουργικότητα και την προσαρμογή στην πρόταση. Υψηλότερες τιμές οδηγούν σε εικόνες που ταιριάζουν πιο κοντά στην πρόταση, ωστόσο πολύ υψηλές τιμές θα επηρεάσουν αρνητικά την ποιότητα. |
| `sampler_name` | `COMBO` | Ο αλγόριθμος που χρησιμοποιείται κατά τη δειγματοληψία, αυτό μπορεί να επηρεάσει την ποιότητα, την ταχύτητα και το στυλ της παραγόμενης έξοδου. |
| `scheduler` | `COMBO` | Ο προγραμματιστής ελέγχει πώς αφαιρείται σταδιακά ο θόρυβος για να σχηματίσει την εικόνα. |
| `denoise` | `FLOAT` | Το ποσό της αποκαθάρισης που εφαρμόζεται, χαμηλότερες τιμές θα διατηρήσουν τη δομή της αρχικής εικόνας επιτρέποντας τη δειγματοληψία εικόνας σε εικόνα. |
| `filename_prefix` | `STRING` | Το πρόθεμα για το αρχείο που θα αποθηκευτεί. Αυτό μπορεί να περιλαμβάνει πληροφορίες μορφοποίησης όπως %date :yyyy-MM-dd% ή %Empty Latent Image.width% για να συμπεριλάβει τιμές από κόμβους. |

### Έξοδοι

| Όνομα | Τύπος | Περιγραφή |
| --- | --- | --- |
| `image` | `IMAGE` | Η αποκωδικοποιημένη εικόνα. |
