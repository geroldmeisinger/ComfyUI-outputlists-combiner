## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow समाविष्ट केलेले)

डिफॉल्ट `CheckpointLoader`, `KSampler`, `VAE Decode` आणि `Save Image` यांचे नोड विस्तार करते जे एकच प्रक्रिया म्हणून प्रक्रिया करते.
हे उपयोगी आहे जर तुम्ही ग्रिडसाठी मध्यवर्ती प्रतिमा तुल्यकालीनपणे सेव करू इच्छित असाल.

*"एक कस्टम KSampler जेणेकरून प्रतिमा सेव करायची? आता मी त्या गोष्टीला बनवले जात आहे जे मी नष्ट करण्याची शोध घेत होतो!"*

### इनपुट्स

| नाव | प्रकार | वर्णन |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | लोड करण्यासाठी चेकपॉइंट (मॉडेल) चे नाव. |
| `positive` | `STRING` | ऑर्डर ज्यामध्ये प्रतिमेत आपल्याला समाविष्ट करायचे असलेले गुण वर्णन करतात. |
| `negative` | `STRING` | ऑर्डर ज्यामध्ये प्रतिमेतून आपल्याला वगळायचे असलेले गुण वर्णन करतात. |
| `latent_image` | `LATENT` | डीनॉइझ करण्यासाठी लॅटेंट प्रतिमा. |
| `seed` | `INT` | शोध निर्माण करण्यासाठी वापरले जाणारे रॅन्डम सीड. |
| `steps` | `INT` | डीनॉइझिंग प्रक्रियेमध्ये वापरलेले स्टेप्सची संख्या. |
| `cfg` | `FLOAT` | Classifier-Free Guidance scale रचना आणि प्रॉम्प्टवर आधारित असलेली संतुलन बनवते. उच्च मूल्ये प्रतिमेत अधिक जुळणारी प्रतिमा देतात हे तर खूप उच्च मूल्ये गुणवत्तेवर नकारात्मक प्रभाव देतात. |
| `sampler_name` | `COMBO` | नमूना करताना वापरले जाणारे अल्गोरिदम, हे उत्पादनाच्या गुणवत्तेवर, वेगावर आणि शैलीवर प्रभाव देऊ शकते. |
| `scheduler` | `COMBO` | स्केज्यूलर शोध असलेल्या शोधाला प्रतिमेत बनवण्यासाठी धीमी धीमी वाढवतो. |
| `denoise` | `FLOAT` | लागू केलेले डीनॉइझिंगची मात्रा, निम्न मूल्ये मूळ प्रतिमेची संरचना ठेवतात ज्यामुळे प्रतिमेतून प्रतिमा नमूना करण्यास सक्षम होतो. |
| `filename_prefix` | `STRING` | सेव करण्यासाठी फाइलचे उपसर्ग. हे फॉरमेटिंग माहिती समाविष्ट करू शकते जसे की %date:yyyy-MM-dd% किंवा %Empty Latent Image.width% जे नोड्समधून मूल्यांना समाविष्ट करतात. |

### आउटपुट्स

| नाव | प्रकार | वर्णन |
| --- | --- | --- |
| `image` | `IMAGE` | डीकोड केलेली प्रतिमा. |
