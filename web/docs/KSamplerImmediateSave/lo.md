## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow ເປັນສ່ວນหนຶ່ງ)

Node ສະຫຼັບຂອງ `CheckpointLoader` ສະເໜີ, `KSampler`, `VAE Decode` ແລະ `Save Image` ທີ່ມີຢູ່ໃນເຄື່ອງມືເລີ່ມຕົ້ນ ເພື່ອດຳເນີນການເປັນໜຶ່ງຄັ້ງ.
ນີ້ເປັນເຄື່ອງມືທີ່ມີຄວາມເປັນເຕັມຖ້າທ່ານຕ້ອງການບັນທຶກຮູບພາບຂອງຮູບແບບເຄື່ອງມືເພື່ອໃຊ້ໃນການສ້າງຮູບແບບຕັ້ງຕົ້ນ.

*"KSampler ສະເໜີໃນການບັນທຶກຮູບພາບ? ນັ້ນຂ້ອຍກໍເປັນສິ່ງທີ່ຂ້ອຍຄົ້ນຫາເພື່ອເປັນເຫດຜົນຂອງມັນ!"*

### ການເຂົ້າຂໍ້ມູນ

| ຊື່ | ພະລົມ | ການອະທິບາຍ |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | ຊື່ຂອງ checkpoint (model) ສຳລັບການໂຫຼດ. |
| `positive` | `STRING` | ການຕັ້ງຄ່າທີ່ສະເໜີໃຫ້ສະເໜີລັກສະນະທີ່ທ່ານຕ້ອງການໃຊ້ໃນຮູບພາບ. |
| `negative` | `STRING` | ການຕັ້ງຄ່າທີ່ສະເໜີໃຫ້ສະເໜີລັກສະນະທີ່ທ່ານຕ້ອງການປະຕິເສດໃນຮູບພາບ. |
| `latent_image` | `LATENT` | ຮູບພາບ latent ສຳລັບການເຮັດໃຫ້ສະເໜີ. |
| `seed` | `INT` | ຄ່າ seed ສຳລັບການສ້າງ noise. |
| `steps` | `INT` | ຈຳນວນຂັ້ນຕອນໃຊ້ໃນຂະບວນການເຮັດໃຫ້ສະເໜີ. |
| `cfg` | `FLOAT` | ຄ່າ scale ສຳລັບ Classifier-Free Guidance ສະເໜີລັກສະນະຂອງຄວາມເປັນເຕັມ ແລະ ການເຂົ້າໃຈຂອງ prompt. ຄ່າສູງຂຶ້ນຈະເຮັດໃຫ້ຮູບພາບມີຄວາມເປັນເຕັມກັບ prompt ແຕ່ຄ່າສູງເກີນໄປຈະເຮັດໃຫ້ຄວາມເປັນເຕັມຂອງຮູບພາບລ່ວມ. |
| `sampler_name` | `COMBO` | ອັລກຳລິດທີ່ໃຊ້ໃນການສ້າງຮູບພາບ, ຄຳນິຍົມນີ້ສາມາດມີຜົນໃຫ້ເກີດຂະບວນການ, ສະເໜີ, ແລະ ສະເໜີໃນຮູບພາບທີ່ສ້າງຂຶ້ນ. |
| `scheduler` | `COMBO` | ການຕັ້ງຄ່າໃຊ້ໃນການເຮັດໃຫ້ noise ສະເໜີເຂົ້າໃນຮູບພາບ. |
| `denoise` | `FLOAT` | ຄ່າສະເໜີໃນການເຮັດໃຫ້ສະເໜີ, ຄ່າຕໍ່ໄປຈະຮັກສາຮູບພາບເດີນໃນຮູບພາບຕົ້ນຕອນ ແລະ ສະເໜີໃນການສ້າງຮູບພາບ. |
| `filename_prefix` | `STRING` | ຊື່ຂອງໄຟລ໌ສຳລັບການບັນທຶກ. ຄຳນິຍົມນີ້ສາມາດມີຂໍ້ມູນເກີດຂະບວນການເຊັ່ນ %date :yyyy-MM-dd% ຫຼື %Empty Latent Image.width% ເພື່ອໃຊ້ໃນຄ່າຈາກ node. |
