## KSampler Ҳамӯҳангӣ Сабт

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow дар ҳамҷоя шудааст)

Кушодани гузоришҳои стандартӣ `CheckpointLoader`, `KSampler`, `VAE Decode` ва `Save Image` барои коркунии якҷоя.
Ин барои корҳои ки шумо мехоҳед, ки суратҳои иҷрошуда барои ҷадвали ҳамҷоя шуда сабт кунед, мавҷуд аст.

*"KSampler-и худи сабт кардани сурат? Амрӯз ман ҳолати, ки ман бо он ҷудо шудам, ҳастам!"*

### Входҳо

| Ном | Тuri | Тавсиф |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | Номи сабт (модел) барои бор кардан. |
| `positive` | `STRING` | Тавсифи шартҳо, ки шумо мехоҳед, ки дар сурат дарҷ шаванд. |
| `negative` | `STRING` | Тавсифи шартҳо, ки шумо мехоҳед, ки дар сурат дарҷ нашаванд. |
| `latent_image` | `LATENT` | Сурати латентӣ барои барқарор кардан. |
| `seed` | `INT` | Кӯшиши худди барқарор кардан барои созиш кардани шумораи шумора. |
| `steps` | `INT` | Шумораи кӯшишҳо, ки дар раванди барқарор кардан истифода мешаванд. |
| `cfg` | `FLOAT` | Манзани ҳамҷоя шудани натиҷаҳо бо барқарор кардани шартҳо барои таъмин кардани ҳамҷоя шудани барқарор кардани шартҳо. Амалҳои баланд ҳамчун суратҳои барқарор кардани шартҳо ба шартҳои сурат барқарор кардан мувофиқ мешаванд, аммо амалҳои баланд манзари суратро барқарор кардани шартҳо ба ҳамҷоя шудани шартҳо манзур мешаванд. |
| `sampler_name` | `COMBO` | Алгоритми истифодашуда дар вақти намудани, ин мумкин аст ба сифат, суръат ва унвони натиҷаҳо таъсир кунад. |
| `scheduler` | `COMBO` | Муҳокимати шумораи шумора барои таъмин кардани шумораи шумораи суратҳо. |
| `denoise` | `FLOAT` | Миқдори барқарор кардани шумора, амалҳои камтар шумораи суратҳои аслӣ ба шумораи суратҳои аслӣ манзур мешаванд, ки имконияти суратҳои аслӣ ба суратҳои аслӣ манзур мешаванд. |
| `filename_prefix` | `STRING` | Префикс барои сабт кардани файл. Ин метавонад тавсифи форматҳо, монанди %date :yyyy-MM-dd% ё %Empty Latent Image.width% бошад, ки амалҳои аз гузоришҳо дар ҳамҷоя шуда манзур мешаванд. |

### Виходҳо

| Ном | Тuri | Тавсиф |
| --- | --- | --- |
| `image` | `IMAGE` | Сурати барқарор карда шуда. |
