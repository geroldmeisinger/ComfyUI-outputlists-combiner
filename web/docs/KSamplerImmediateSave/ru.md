## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow включен)

Расширение узла для стандартных узлов `CheckpointLoader`, `KSampler`, `VAE Decode` и `Save Image`, чтобы обрабатывать их как один блок.
Это полезно, если вы хотите сохранить промежуточные изображения для сетки немедленно.

*"Пользовательский KSampler только чтобы сохранить изображение? Теперь я стал тем, что хотел уничтожить!"*

### Входы

| Имя | Тип | Описание |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | Имя чекпоинта (модели) для загрузки. |
| `positive` | `STRING` | Условие, описывающее атрибуты, которые вы хотите включить в изображение. |
| `negative` | `STRING` | Условие, описывающее атрибуты, которые вы хотите исключить из изображения. |
| `latent_image` | `LATENT` | Латентное изображение для денойзинга. |
| `seed` | `INT` | Случайное семя, используемое для создания шума. |
| `steps` | `INT` | Количество шагов, используемых в процессе денойзинга. |
| `cfg` | `FLOAT` | Масштаб без классификаторного руководства балансирует между креативностью и приверженностью к промпту. Более высокие значения приводят к изображениям, более точно соответствующим промпту, однако слишком высокие значения негативно повлияют на качество. |
| `sampler_name` | `COMBO` | Алгоритм, используемый при выборке, может влиять на качество, скорость и стиль генерируемого результата. |
| `scheduler` | `COMBO` | Планировщик контролирует, как постепенно удаляется шум для формирования изображения. |
| `denoise` | `FLOAT` | Количество денойзинга, применяемого, более низкие значения сохранят структуру исходного изображения, позволяя выполнять sampling изображение в изображение. |
| `filename_prefix` | `STRING` | Префикс для файла для сохранения. Это может включать информацию о форматировании, такую как %date :yyyy-MM-dd% или %Empty Latent Image.width% для включения значений из узлов. |

### Выходы

| Имя | Тип | Описание |
| --- | --- | --- |
| `image` | `IMAGE` | Декодированное изображение. |
