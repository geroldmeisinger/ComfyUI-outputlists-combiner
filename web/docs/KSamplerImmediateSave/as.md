## KSampler Immediate Save

![KSampler Immediate Save](KSamplerImmediateSave/KSamplerImmediateSave.png)

(ComfyUI workflow সমাহিত)

ডিফল্ট `CheckpointLoader`, `KSampler`, `VAE Decode` আৰু `Save Image` নোড বিস্তাৰ কৰে যাতে একেটা কাৰ্য্য পৰিচালনা কৰিব।
এইটো গৰিড বাবে মধ্যবর্তী চিত্ৰ সংৰক্ষণ কৰিবলৈ আপুনি বিচাৰে তেনে ক্ষেত্ৰত উপকাৰী।

*"এটা চিত্ৰ সংৰক্ষণ কৰিবলৈ কাষ্টম KSampler? এতিয়া মই নিজেকে ধৰি সংৰক্ষণ কৰিবলৈ যোৱা বস্তু হৈ গলিছো!"*

### ইনপুট

| নাম | প্ৰকাৰ | বিবৰণ |
| --- | --- | --- |
| `cpkt_name` | `COMBO` | লোড কৰিবলৈ পাৰ্শ্বস্থান (মডেল) নাম। |
| `positive` | `STRING` | চিত্ৰত সম্মিলিত কৰিবলৈ আপুনি বিচাৰে বৈশিষ্ট্যসমূহ বর্ণনা কৰে। |
| `negative` | `STRING` | চিত্ৰত বাদ দিবলৈ আপুনি বিচাৰে বৈশিষ্ট্যসমূহ বর্ণনা কৰে। |
| `latent_image` | `LATENT` | ডিনয়োইজ কৰিবলৈ পাৰ্শ্বস্থান চিত্ৰ। |
| `seed` | `INT` | শব্দ সৃষ্টি কৰিবলৈ ব্যৱহৃত ক্ষুদ্ৰ বৰ্ণনা। |
| `steps` | `INT` | ডিনয়োইজিং প্ৰক্ৰিয়াত ব্যৱহৃত পদসমূহৰ সংখ্যা। |
| `cfg` | `FLOAT` | Classifier-Free Guidance স্কেল সৃজনশীলতা আৰু প্ৰম্প্টত অনুসৰণৰ মাজত সন্তুলন কৰে। উচ্চ মান প্ৰম্প্টৰ সৈতে চিত্ৰ বেছিকৈ মেলে তেনে ক্ষেত্ৰত হৈ থাকে কিন্তু খুব উচ্চ মানে মান প্ৰভাৱ নেকি হৈ থাকে। |
| `sampler_name` | `COMBO` | নমুনা কৰিবলৈ ব্যৱহৃত অ্যালগৰিদম, এইটো উৎপাদনৰ মান, গতি আৰু শৈলীত প্ৰভাৱ ফেলোৱা পাৰে। |
| `scheduler` | `COMBO` | স্কেজুলাৰ শব্দ সৃষ্টি কৰিবলৈ ব্যৱহৃত হৈ থাকে যাতে চিত্ৰ সৃষ্টি কৰা হয়। |
| `denoise` | `FLOAT` | ডিনয়োইজিং প্ৰয়োগ কৰা পৰিমাণ, কম মান মূল চিত্ৰৰ গঠন বজাই থাকে যাতে চিত্ৰ-পৰিবৰ্তন কৰা যায়। |
| `filename_prefix` | `STRING` | সংৰক্ষণ কৰা ফাইলৰ উপসৰ্গ। এইটো ফরমেটিং তথ্য অন্তৰ্ভুক্ত কৰিব পাৰে যেনে %date :yyyy-MM-dd% অথবা %Empty Latent Image.width% যাতে নোড পৰিমাণ অন্তৰ্ভুক্ত কৰা যায়। |

### আউটপুট

| নাম | প্ৰকাৰ | বিবৰণ |
| --- | --- | --- |
| `image` | `IMAGE` | ডিকোড কৰা চিত্ৰ। |
