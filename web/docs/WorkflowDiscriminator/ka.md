## Workflow Discriminator

![Workflow Discriminator](WorkflowDiscriminator/WorkflowDiscriminator.png)

(ComfyUI рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ)

рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃА рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃА рЃЊрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЏрЃљрЃЌ рЃўрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃЮрЃА рЃўрЃюрЃЊрЃўрЃЋрЃўрЃЊрЃБрЃљрЃџрЃБрЃарЃў OutputLists-рЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ.
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃћрЃА рЃюрЃЮрЃЊрЃў рЃўрЃЏрЃўрЃА рЃљрЃдрЃЊрЃњрЃћрЃюрЃљрЃерЃў, рЃЌрЃБ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃа рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃЏрЃљрЃЋрЃћ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃАрЃўрЃўрЃЊрЃљрЃю.
рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ ComfyUI-рЃА `IMAGE` рЃљрЃа рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃўрЃА рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЊрЃљ рЃЌрЃЦрЃЋрЃћрЃю рЃБрЃюрЃЊрЃљ рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЮрЃЌ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃўрЃА+рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃћрЃЉрЃўрЃЌ рЃЊрЃљ рЃЊрЃљрЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЮрЃЌ рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃљрЃЏ рЃюрЃЮрЃЊрЃА.
рЃЏрЃћрЃбрЃљрЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃбрЃЋрЃўрЃарЃЌрЃљрЃЋрЃћрЃЉрЃўрЃЌ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃюрЃЮрЃЊрЃћрЃЉрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА:
* `Load Any File.metadata` -> `JSON OutputList(jsonpath=$.["PNG:Prompt"]).value`
* [Crystool](https://github.com/crystian/ComfyUI-Crystools) `­ЪфЏ Load image with metadata.Metadata RAW` -> `­ЪфЏ Metadata extractor.prompt`
* [Simple_Readable_Metadata](https://github.com/ShammiG/ComfyUI-Simple_Readable_Metadata-SG) `Simple Readable Metadata-SG.metadata_raw`
